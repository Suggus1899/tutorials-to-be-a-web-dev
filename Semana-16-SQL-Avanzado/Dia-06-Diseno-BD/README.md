# Día 6: Diseño de Bases de Datos

En este día aprenderás buenas prácticas de diseño de BD.

## Contenido
- Proceso de diseño
- Diagramas ER
- Naming conventions
- Mejores prácticas

## Tareas

### Tarea 1: Diagrama ER
Diseña un diagrama ER para:
- Sistema de biblioteca (libros, autores, préstamos, usuarios)
- Usa herramientas como draw.io o dbdiagram.io

### Tarea 2: Naming conventions
Aplica convenciones:
- snake_case para nombres (user_id, created_at)
- Tablas en plural (users, products)
- Columnas descriptivas
- Prefijos consistentes para FKs

### Tarea 3: Schema completo
Crea un schema SQL completo para el diagrama ER:
- Todas las tablas con tipos correctos
- Primary y Foreign keys
- Constraints apropiados
- Índices básicos

## Ejercicios Adicionales
- Revisa schemas de proyectos open source
- Practica con índices para optimización
