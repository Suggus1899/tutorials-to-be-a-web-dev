# Semana 16: SQL Avanzado

## Objetivos de la Semana

Esta semana aprenderás conceptos avanzados de SQL:

### 1. Joins
- INNER JOIN
- LEFT JOIN, RIGHT JOIN
- Relaciones entre tablas

### 2. Relaciones
- One-to-Many
- Many-to-Many
- Foreign Keys

### 3. Normalización
- Formas normales
- Evitar redundancia
- Diseño de bases de datos

## Estructura de la Semana

- **Día 1-2**: Joins y combinación de tablas
- **Día 3-4**: Relaciones entre entidades
- **Día 5-6**: Normalización y diseño
