# Día 2: Tipos de JOINs

En este día aprenderás los diferentes tipos de JOIN.

## Contenido
- INNER JOIN
- LEFT JOIN
- RIGHT JOIN (si está disponible)
- CROSS JOIN

## Tareas

### Tarea 1: LEFT JOIN
Practica LEFT JOIN:
- Obtén todos los autores, incluso sin libros
- Observa los valores NULL
- Compara resultados con INNER JOIN

### Tarea 2: Diferencia entre JOINs
Crea escenarios que muestren la diferencia:
- Autor sin libros
- Libro sin autor (si es posible)
- Observa qué registros incluye cada JOIN

### Tarea 3: JOINs prácticos
Implementa consultas reales:
- Lista de productos con categoría (LEFT JOIN)
- Pedidos con cliente (INNER JOIN)
- Empleados con departamento

## Ejercicios Adicionales
- Investiga FULL OUTER JOIN
- Practica con self JOINs (tabla consigo misma)
