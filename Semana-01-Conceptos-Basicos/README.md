# Semana 1: Conceptos Básicos

## Objetivos de la Semana

Esta semana aprenderás los fundamentos esenciales de la programación:

### 1. Conceptos Fundamentales
- Comprender qué es un algoritmo
- Entender la lógica de programación
- Conocer los paradigmas básicos

### 2. Instalación y Configuración del Entorno
- Configurar el entorno de desarrollo
- Instalar las herramientas necesarias
- Familiarizarse con el editor de código

### 3. Variables
- Declaración de variables
- Ámbito de variables (scope)
- Mejores prácticas para nombrar variables

### 4. Tipos de Datos
- Tipos primitivos
- Tipos compuestos
- Conversión entre tipos

### 5. Operadores
- Operadores aritméticos
- Operadores de asignación
- Precedencia de operadores
- Operadores lógicos y booleanos

## Estructura de la Semana

- **Día 1**: Algoritmos
- **Día 2**: Entorno de desarrollo
- **Día 3**: Variables
- **Día 4**: Tipos de datos
- **Día 5**: Operadores aritméticos
- **Día 6**: Lógica booleana
- **Día 7**: Práctica integradora
