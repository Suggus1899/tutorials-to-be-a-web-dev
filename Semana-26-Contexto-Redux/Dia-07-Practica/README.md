## Proyecto del Día

### Proyecto: E-commerce con Estado Global

Crea una tienda con manejo de estado completo:

1. **Redux Slices:**
   - productsSlice: lista de productos
   - cartSlice: carrito de compras
   - userSlice: autenticación
   - uiSlice: estado de UI (modals, toasts)

2. **Async Actions:**
   - Fetch products
   - Login/logout
   - Checkout

3. **Features:**
   - Lista de productos
   - Agregar al carrito
   - Ver carrito
   - Checkout
   - Autenticación

4. **Selectors:**
   - Total del carrito
   - Cantidad de items
   - Productos filtrados

5. **Extras:**
   - Redux DevTools
   - Persistencia del carrito
   - Optimistic updates
