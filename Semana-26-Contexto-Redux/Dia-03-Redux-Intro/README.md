# Día 3: Introducción a Redux

En este día aprenderás los fundamentos de Redux.

## Contenido
- Qué es Redux
- Store, Actions, Reducers
- Flujo unidireccional
- Cuándo usar Redux

## Tareas

### Tarea 1: Conceptos de Redux
Estudia los conceptos:
- Store: estado global
- Actions: describen qué pasó
- Reducers: cómo cambia el estado
- Dispatch: envía acciones

### Tarea 2: Flujo de datos
Entiende el flujo:
1. Componente dispara action
2. Store ejecuta reducer
3. Reducer retorna nuevo estado
4. Componentes se re-renderizan

### Tarea 3: Cuándo usar Redux
Analiza:
- Estado compartido por muchos componentes
- Estado que cambia frecuentemente
- Aplicaciones grandes
- Necesidad de debugging avanzado
