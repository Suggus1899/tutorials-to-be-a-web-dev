# Semana 10: DOM y Eventos

## Objetivos de la Semana

Esta semana aprenderás a manipular el DOM y manejar eventos:

### 1. querySelector
- Seleccionar elementos del DOM
- querySelector vs querySelectorAll
- Diferencias con getElementById

### 2. Manipulación del DOM
- Crear, modificar y eliminar elementos
- Modificar atributos y clases
- innerHTML vs textContent

### 3. Event Listeners
- addEventListener
- Tipos de eventos
- Event object y propagación

## Estructura de la Semana

- **Día 1-2**: querySelector y selección de elementos
- **Día 3-4**: Manipulación del DOM
- **Día 5-6**: Event listeners y manejo de eventos
- **Día 7**: Práctica integradora
