# Día 1: querySelector

En este día aprenderás a seleccionar elementos del DOM con querySelector.

## Contenido
- querySelector() vs getElementById()
- querySelectorAll()
- Selectores CSS en JavaScript
- NodeList vs HTMLCollection

## Tareas

### Tarea 1: Selecciones básicas
Practica seleccionando:
- Un elemento por ID
- Un elemento por clase
- Un elemento por etiqueta

### Tarea 2: Selectores avanzados
Usa selectores CSS complejos:
- Selector de descendientes (.container p)
- Selector de atributos ([type="text"])
- Pseudo-clases (:first-child, :nth-child)

### Tarea 3: querySelectorAll
Practica con múltiples elementos:
- Selecciona todos los elementos de una clase
- Itera sobre los resultados con forEach
- Convierte NodeList a array

## Ejercicios Adicionales
- Compara rendimiento entre métodos
- Experimenta con closest() para buscar hacia arriba
