# Día 5: Event Listeners

En este día aprenderás a manejar eventos con addEventListener.

## Contenido
- addEventListener()
- removeEventListener()
- Tipos de eventos comunes
- El objeto Event

## Tareas

### Tarea 1: Eventos de mouse
Practica con:
- click
- dblclick
- mouseenter / mouseleave
- mousemove

### Tarea 2: Eventos de teclado
Practica con:
- keydown / keyup
- Obtener la tecla presionada (event.key)
- Detectar teclas especiales (Ctrl, Shift)

### Tarea 3: Eventos de formulario
Practica con:
- submit (y preventDefault)
- input / change
- focus / blur

## Ejercicios Adicionales
- Experimenta con el objeto event y sus propiedades
- Practica removiendo event listeners
