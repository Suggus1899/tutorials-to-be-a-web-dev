# Día 3: Crear Elementos

En este día aprenderás a crear elementos del DOM dinámicamente.

## Contenido
- createElement()
- createTextNode()
- appendChild() y append()
- insertBefore() y insertAdjacentElement()

## Tareas

### Tarea 1: Crear elementos simples
Practica:
- Crear un párrafo y agregarlo al body
- Crear un enlace con atributos
- Crear una imagen con src y alt

### Tarea 2: Crear estructuras complejas
Crea dinámicamente:
- Una lista (ul/li) desde un array de datos
- Una card con título, descripción e imagen
- Una tabla desde un array de objetos

### Tarea 3: Diferentes métodos de inserción
Practica:
- insertBefore para insertar antes de un elemento
- insertAdjacentElement con las 4 posiciones
- replaceChild para reemplazar elementos

## Ejercicios Adicionales
- Compara append() vs appendChild()
- Experimenta con DocumentFragment para mejor rendimiento
