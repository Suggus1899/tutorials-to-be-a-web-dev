# Día 6: Eventos Avanzados

En este día aprenderás conceptos avanzados de manejo de eventos.

## Contenido
- Event bubbling y capturing
- Event delegation
- stopPropagation y preventDefault
- Custom events

## Tareas

### Tarea 1: Entender bubbling
Practica:
- Crea elementos anidados con eventos
- Observa cómo el evento burbujea hacia arriba
- Usa stopPropagation cuando sea necesario

### Tarea 2: Event delegation
Implementa:
- Un listener en el contenedor padre
- Detecta qué elemento hijo fue clickeado
- Agrega elementos dinámicamente que respondan a eventos

### Tarea 3: Casos prácticos
Implementa:
- Una lista donde cada item es clickeable
- Un formulario con validación en submit
- Un botón que cambie de estado

## Ejercicios Adicionales
- Experimenta con captura (useCapture: true)
- Crea y dispara eventos personalizados (CustomEvent)
