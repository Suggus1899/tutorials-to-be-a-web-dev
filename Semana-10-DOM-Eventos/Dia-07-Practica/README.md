- Proyecto integrador

## Proyecto del Día

### Proyecto: Lista de Tareas Interactiva (Todo List)

Crea una aplicación de lista de tareas:

1. **Estructura HTML:**
   - Input para nueva tarea
   - Botón para agregar
   - Lista de tareas
   - Filtros (todas, completadas, pendientes)

2. **Funcionalidades:**
   - Agregar tarea (crear elemento y agregarlo al DOM)
   - Marcar como completada (toggle de clase)
   - Eliminar tarea (remover del DOM)
   - Filtrar tareas (mostrar/ocultar)
   - Contar tareas pendientes

3. **Eventos a implementar:**
   - Submit del formulario
   - Click en checkbox para completar
   - Click en botón eliminar
   - Click en filtros
   - Enter para agregar tarea

4. **Optimizaciones:**
   - Usa event delegation para los items
   - Valida que la tarea no esté vacía
   - Permite editar tareas existentes

## Tareas Adicionales

### Tarea 1: Persistencia
Guarda las tareas en localStorage.

### Tarea 2: Animaciones
Agrega animaciones CSS al agregar/eliminar tareas.
