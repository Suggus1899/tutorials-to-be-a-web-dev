# Día 2: Selección de Elementos

En este día aprenderás diferentes métodos para seleccionar elementos.

## Contenido
- getElementById
- getElementsByClassName
- getElementsByTagName
- Navegación del DOM (parent, children, siblings)

## Tareas

### Tarea 1: Métodos de selección
Practica con cada método:
- Selecciona por ID
- Selecciona por clase (colección viva)
- Selecciona por etiqueta

### Tarea 2: Navegación del DOM
Navega entre elementos:
- parentElement / parentNode
- children / childNodes
- nextElementSibling / previousElementSibling
- firstElementChild / lastElementChild

### Tarea 3: Combinar selección y navegación
Practica:
- Selecciona un elemento y navega a su padre
- Selecciona un contenedor y accede a sus hijos
- Recorre todos los hermanos de un elemento

## Ejercicios Adicionales
- Experimenta con la diferencia entre children y childNodes
- Practica con matches() para verificar selectores
