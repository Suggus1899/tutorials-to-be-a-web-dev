# Día 4: Modificar el DOM

En este día aprenderás a modificar elementos existentes del DOM.

## Contenido
- innerHTML vs textContent
- Modificar atributos (setAttribute, getAttribute)
- Modificar clases (classList)
- Modificar estilos (style)

## Tareas

### Tarea 1: Modificar contenido
Practica:
- Cambiar el texto de un elemento con textContent
- Cambiar el HTML interno con innerHTML
- Agregar contenido sin reemplazar

### Tarea 2: Modificar atributos y clases
Practica:
- Agregar/quitar clases con classList.add/remove
- Toggle de clases con classList.toggle
- Cambiar atributos con setAttribute

### Tarea 3: Modificar estilos
Practica:
- Cambiar estilos individuales con element.style
- Agregar múltiples estilos
- Usar cssText para estilos en línea

## Ejercicios Adicionales
- Investiga la diferencia entre textContent e innerText
- Practica con dataset para atributos data-*
