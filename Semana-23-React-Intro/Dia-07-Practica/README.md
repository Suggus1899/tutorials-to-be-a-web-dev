## Proyecto del Día

### Proyecto: Galería de Productos

Crea una galería de productos:

1. **Componentes a crear:**
   - Header con título
   - ProductCard con imagen, nombre, precio
   - ProductList que renderiza lista
   - Footer

2. **Datos:**
   ```javascript
   const products = [
     { id: 1, name: 'Laptop', price: 999, image: '...' },
     { id: 2, name: 'Mouse', price: 29, image: '...' },
   ];
   ```

3. **Funcionalidades:**
   - Mostrar lista de productos
   - Cada producto en una card
   - Botón "Ver detalles" en cada card
   - Mostrar alert con info del producto

4. **Estilos:**
   - Grid de productos
   - Cards con hover effect
   - Diseño responsive
