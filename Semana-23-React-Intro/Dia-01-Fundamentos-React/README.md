# Día 1: Fundamentos de React

En este día aprenderás los fundamentos de React.

## Contenido
- Qué es React
- Virtual DOM
- Configuración con Vite
- Estructura de proyecto

## Tareas

### Tarea 1: Crear proyecto React
Usa Vite para crear proyecto:
```bash
npm create vite@latest mi-app -- --template react
cd mi-app
npm install
npm run dev
```

### Tarea 2: Explorar la estructura
Entiende los archivos:
- index.html: punto de entrada
- main.jsx: monta React
- App.jsx: componente principal
- package.json: dependencias

### Tarea 3: Virtual DOM
Investiga:
- Qué es el Virtual DOM
- Por qué React lo usa
- Reconciliation
