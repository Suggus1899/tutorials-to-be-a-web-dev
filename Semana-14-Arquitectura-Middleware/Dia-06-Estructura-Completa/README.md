# Día 6: Estructura Completa

En este día implementarás una estructura de proyecto completa.

## Contenido
- Estructura de carpetas profesional
- Configuración centralizada
- Manejo de errores robusto
- Logging y debugging

## Tareas

### Tarea 1: Estructura de carpetas
Crea esta estructura:
```
src/
  config/
  controllers/
  middlewares/
  routes/
  services/
  utils/
  app.js
  server.js
```

### Tarea 2: Configuración centralizada
Implementa:
- Archivo de configuración (config/index.js)
- Variables de entorno organizadas
- Configuración por ambiente (dev, prod)

### Tarea 3: Archivo app.js organizado
Organiza app.js:
- Importaciones ordenadas
- Middlewares globales
- Rutas montadas
- Error handlers al final

## Ejercicios Adicionales
- Agrega scripts de desarrollo (nodemon)
- Implementa un health check endpoint
