## Proyecto del Día

### Proyecto: API de E-commerce con Arquitectura Limpia

Crea una API con arquitectura de capas:

1. **Estructura del proyecto:**
   ```
   src/
     config/
     controllers/
     middlewares/
     routes/
     services/
     utils/
     app.js
   ```

2. **Recursos a implementar:**
   - Productos (CRUD completo)
   - Categorías (CRUD completo)
   - Carrito de compras

3. **Middlewares a crear:**
   - Logger de requests
   - Validación de datos
   - Manejo de errores global
   - Autenticación simulada

4. **Servicios:**
   - productService con lógica de negocio
   - categoryService
   - cartService

5. **Características:**
   - Separación clara de responsabilidades
   - Respuestas consistentes
   - Manejo de errores robusto
   - Código limpio y organizado
