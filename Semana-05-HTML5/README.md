# Semana 5: HTML5

## Objetivos de la Semana

Esta semana aprenderás la estructura del web con HTML5:

### 1. Estructura del DOM
- Qué es el DOM
- Elementos y nodos
- Jerarquía de elementos

### 2. Formularios
- Elementos de formulario
- Tipos de input
- Validación básica

### 3. Etiquetas semánticas
- header, nav, main, footer
- article, section, aside
- Importancia de la semántica

## Estructura de la Semana

- **Día 1-2**: Estructura del DOM y elementos básicos
- **Día 3-4**: Formularios y validación
- **Día 5-6**: Etiquetas semánticas
- **Día 7**: Práctica integradora
