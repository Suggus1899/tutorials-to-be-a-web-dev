# Día 3: Formularios Básico

En este día aprenderás a crear formularios en HTML.

## Contenido
- Etiqueta form y atributos básicos
- Tipos de input (text, email, password, number)
- Labels y accesibilidad
- Botones submit y reset

## Tareas

### Tarea 1: Formulario de contacto
Crea un formulario con:
- Campos para nombre, email y mensaje
- Labels asociados correctamente
- Botón de enviar

### Tarea 2: Formulario de registro
Crea un formulario de registro con:
- Username
- Email
- Password y confirmación de password
- Checkbox para términos y condiciones

### Tarea 3: Diferentes tipos de input
Crea una página que demuestre todos los tipos de input:
- text, email, password, number
- date, time, color
- range, checkbox, radio

## Ejercicios Adicionales
- Experimenta con el atributo placeholder
- Practica agrupando campos con fieldset y legend
