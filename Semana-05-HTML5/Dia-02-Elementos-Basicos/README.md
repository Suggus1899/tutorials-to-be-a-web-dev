# Día 2: Elementos Básicos

En este día aprenderás los elementos HTML más comunes.

## Contenido
- Encabezados (h1-h6)
- Párrafos y texto
- Enlaces e imágenes
- Listas ordenadas y no ordenadas

## Tareas

### Tarea 1: Página de presentación personal
Crea una página con:
- Un encabezado h1 con tu nombre
- Párrafos describiendo quién eres
- Una imagen (puede ser un placeholder)
- Una lista de tus hobbies

### Tarea 2: Página de enlaces
Crea una página con enlaces a:
- Sitios externos (target="_blank")
- Secciones dentro de la misma página (anchor links)
- Otra página HTML en tu proyecto

### Tarea 3: Página con listas
Crea una página con:
- Una lista ordenada de pasos para una receta
- Una lista no ordenada de ingredientes
- Una lista anidada

## Ejercicios Adicionales
- Experimenta con etiquetas de formato (strong, em, mark)
- Practica con la etiqueta figure y figcaption
