# Día 6: Accesibilidad

En este día aprenderás buenas prácticas de accesibilidad en HTML.

## Contenido
- Atributos ARIA básicos
- Alt text para imágenes
- Skip links
- Orden de tabulación

## Tareas

### Tarea 1: Revisar accesibilidad
Revisa las páginas que has creado y asegúrate de que:
- Todas las imágenes tienen alt text descriptivo
- Los formularios tienen labels correctos
- La estructura de encabezados es correcta

### Tarea 2: Agregar ARIA
Mejora la accesibilidad de una página agregando:
- aria-label donde sea necesario
- aria-describedby para campos de formulario
- role para elementos interactivos

### Tarea 3: Navegación por teclado
Verifica que tu página sea navegable por teclado:
- Todos los elementos interactivos son accesibles
- El orden de tabulación es lógico
- Los estados de focus son visibles

## Ejercicios Adicionales
- Usa un lector de pantalla para probar tu página
- Ejecuta un análisis de accesibilidad con Lighthouse
