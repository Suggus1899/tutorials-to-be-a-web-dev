# Día 1: Estructura del DOM

En este día aprenderás qué es el DOM y la estructura básica de un documento HTML.

## Contenido
- Qué es el DOM (Document Object Model)
- Estructura básica de HTML5
- Elementos y nodos
- Jerarquía de elementos

## Tareas

### Tarea 1: Crear un documento HTML básico
Crea un archivo HTML con la estructura básica: `<!DOCTYPE html>`, `<html>`, `<head>`, `<body>`.

### Tarea 2: Agregar metadatos
Agrega las etiquetas meta esenciales: charset, viewport, title y description.

### Tarea 3: Explorar el DOM
Abre las DevTools del navegador y explora la estructura del DOM de tu página.

## Ejercicios Adicionales
- Investiga sobre la diferencia entre DOM y HTML
- Practica inspeccionando el DOM de páginas web populares
