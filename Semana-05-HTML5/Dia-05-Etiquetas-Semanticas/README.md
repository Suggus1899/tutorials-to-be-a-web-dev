# Día 5: Etiquetas Semánticas

En este día aprenderás sobre las etiquetas semánticas de HTML5.

## Contenido
- header, nav, main, footer
- article, section, aside
- Importancia del HTML semántico
- SEO y accesibilidad

## Tareas

### Tarea 1: Estructura semántica
Crea una página con la estructura semántica completa:
- Header con logo y navegación
- Main con secciones de contenido
- Footer con información de contacto

### Tarea 2: Blog post
Crea una página de blog usando:
- article para el post
- section para diferentes partes del contenido
- aside para contenido relacionado

### Tarea 3: Página de portfolio
Crea una página de portfolio usando todas las etiquetas semánticas apropiadas.

## Ejercicios Adicionales
- Investiga sobre el outline del documento HTML
- Practica usando las herramientas de accesibilidad del navegador
