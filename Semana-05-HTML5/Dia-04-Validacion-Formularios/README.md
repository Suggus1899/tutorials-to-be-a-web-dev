# Día 4: Validación de Formularios

En este día aprenderás a validar formularios con HTML5.

## Contenido
- Atributos de validación (required, pattern, min, max)
- Mensajes de error nativos
- Pseudo-clases de validación (:valid, :invalid)
- Atributo novalidate

## Tareas

### Tarea 1: Validación requerida
Modifica tus formularios anteriores para que todos los campos obligatorios tengan el atributo `required`.

### Tarea 2: Validación con patrones
Crea un campo que valide:
- Un número de teléfono con un patrón específico
- Un código postal
- Un username con solo letras y números

### Tarea 3: Validación de rangos
Crea un formulario con:
- Campo de edad con min="18" y max="100"
- Campo de cantidad con restricciones
- Campo de texto con minlength y maxlength

## Ejercicios Adicionales
- Investiga cómo personalizar mensajes de error
- Practica combinando múltiples validaciones
