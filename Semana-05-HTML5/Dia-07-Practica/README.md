# Día 7: Práctica Integradora

En este día realizarás ejercicios que integran todos los conceptos de la semana.

## Contenido
- Repaso de estructura HTML
- Repaso de formularios
- Repaso de semántica y accesibilidad
- Proyecto integrador

## Proyecto del Día

### Proyecto: Página Web Personal Completa

Crea una página web personal que incluya:

1. **Estructura semántica:**
   - Header con navegación
   - Main con múltiples secciones
   - Footer con información de contacto

2. **Secciones requeridas:**
   - Inicio/Hero con tu nombre y título
   - Sobre mí con descripción personal
   - Habilidades con lista de tecnologías
   - Proyectos con cards de proyectos
   - Contacto con formulario

3. **Formulario de contacto:**
   - Campos: nombre, email, asunto, mensaje
   - Validación HTML5 apropiada
   - Labels y accesibilidad correcta

4. **Accesibilidad:**
   - Alt text en todas las imágenes
   - Estructura de encabezados correcta
   - Navegable por teclado

## Tareas Adicionales

### Tarea 1: Agregar más secciones
Agrega una sección de educación o experiencia laboral.

### Tarea 2: Validar HTML
Usa el validador de W3C para verificar que tu HTML es válido.
