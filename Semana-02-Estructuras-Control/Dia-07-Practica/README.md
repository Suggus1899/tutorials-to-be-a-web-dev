## Proyecto del Día

### Proyecto: Sistema de Evaluación de Estudiantes

Crea un programa que:

1. **Reciba información del estudiante:**
   - Nombre
   - Calificación del examen (0-100)
   - Número de asistencias (0-20)

2. **Calcule el estado del estudiante usando switch:**
   - "Aprobado": calificación >= 70 y asistencias >= 15
   - "Recuperación": calificación >= 50 y < 70
   - "Reprobado": calificación < 50 o asistencias < 10
   - "Pendiente": otros casos

3. **Muestre un mensaje personalizado:**
   - Usa condicionales para mostrar recomendaciones según el estado

4. **Implementa debugging:**
   - Agrega console.log para seguir el flujo
   - Maneja casos de datos inválidos
