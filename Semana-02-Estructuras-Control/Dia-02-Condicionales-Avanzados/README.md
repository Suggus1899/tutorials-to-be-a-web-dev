# Día 2: Condicionales Avanzados

En este día aprenderás sobre condicionales anidados y el operador ternario.

## Contenido
- Condicionales anidados
- Operador ternario
- Operadores lógicos (&&, ||, !)
- Mejores prácticas

## Tareas

### Tarea 1: Sistema de calificaciones
Crea un programa que reciba una calificación numérica (0-100) y retorne la letra correspondiente:
- A: 90-100
- B: 80-89
- C: 70-79
- D: 60-69
- F: menos de 60

### Tarea 2: Verificar año bisiesto
Crea un programa que determine si un año es bisiesto o no.

### Tarea 3: Convertir a operador ternario
Toma los ejercicios del Día 1 y reescríbelos usando el operador ternario.

## Ejercicios Adicionales
- Practica combinando operadores lógicos
- Experimenta con condicionales anidados de múltiples niveles
