# Día 4: Switch - Casos de Uso

En este día aprenderás casos de uso avanzados del switch.

## Contenido
- Casos múltiples
- Switch con strings
- Switch vs objeto de lookup
- Cuándo usar switch vs if/else

## Tareas

### Tarea 1: Calculadora básica
Crea una calculadora que reciba dos números y un operador (+, -, *, /) y retorne el resultado.

### Tarea 2: Estaciones del año
Crea un programa que reciba un mes y retorne la estación del año correspondiente. Usa casos múltiples para agrupar meses.

### Tarea 3: Traductor de colores
Crea un programa que reciba el nombre de un color en español y retorne su traducción al inglés (usa al menos 5 colores).

## Ejercicios Adicionales
- Investiga y practica el patrón de objeto de lookup como alternativa a switch
- Reescribe uno de los ejercicios usando if/else y compara la legibilidad
