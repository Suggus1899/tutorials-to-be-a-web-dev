# Día 5: Debugging Básico

En este día aprenderás técnicas básicas de debugging.

## Contenido
- Uso de console.log
- console.warn y console.error
- console.table para datos estructurados
- Lectura de mensajes de error

## Tareas

### Tarea 1: Depurar con console.log
Toma un programa con condicionales del Día 1 o 2 y agrega console.log para seguir el flujo de ejecución.

### Tarea 2: Identificar errores comunes
Analiza el siguiente código y encuentra los errores:
```javascript
let edad = "25";
if (edad = 18) {
    console.log("Es mayor de edad");
}
```

### Tarea 3: Usar console.table
Crea un array de objetos y usa console.table para visualizarlo de forma clara.

## Ejercicios Adicionales
- Practica leyendo y entendiendo mensajes de error del navegador
- Experimenta con console.group para organizar mensajes
