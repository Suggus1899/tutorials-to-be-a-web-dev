# Día 1: Condicionales if/else

En este día aprenderás sobre las estructuras condicionales básicas en JavaScript.

## Contenido
- Sintaxis de if/else
- Condicionales simples
- Operadores de comparación
- Ejemplos prácticos

## Tareas

### Tarea 1: Verificar si un número es positivo o negativo
Crea un programa que reciba un número y determine si es positivo, negativo o cero.

### Tarea 2: Determinar si una persona es mayor de edad
Crea un programa que reciba la edad de una persona y determine si es mayor de edad (18 años o más).

### Tarea 3: Comparar dos números
Crea un programa que reciba dos números y determine cuál es mayor, o si son iguales.

## Ejercicios Adicionales
- Practica escribiendo diferentes condiciones
- Experimenta con operadores de comparación (==, ===, >, <, >=, <=)
