# Semana 2: Estructuras de Control

## Objetivos de la Semana

Esta semana aprenderás a controlar el flujo de tus programas:

### 1. Condicionales if/else
- Sintaxis de if/else
- Condicionales anidados
- Operador ternario

### 2. Switch
- Sintaxis de switch
- Casos múltiples
- Default case

### 3. Debugging
- Uso de console.log
- Herramientas de debugging
- Estrategias para encontrar errores

## Estructura de la Semana

- **Día 1-2**: Condicionales if/else
- **Día 3-4**: Switch y casos de uso
- **Día 5-6**: Debugging y resolución de problemas
- **Día 7**: Práctica integradora
