# Semana 8: Arrays y Objetos

## Objetivos de la Semana

Esta semana aprenderás a trabajar con estructuras de datos complejas:

### 1. Arrays
- Acceso a elementos
- Métodos de arrays (push, pop, shift, unshift)
- Iteración sobre arrays

### 2. Objetos
- Propiedades y métodos
- Acceso con notación punto y corchetes
- this y contexto

### 3. Destructuring
- Destructuring de arrays
- Destructuring de objetos
- Rest y spread operators

## Estructura de la Semana

- **Día 1-2**: Arrays y sus métodos
- **Día 3-4**: Objetos y propiedades
- **Día 5-6**: Destructuring y operadores modernos
- **Día 7**: Práctica integradora
