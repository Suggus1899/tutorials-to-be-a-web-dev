# Día 5: Destructuring de Arrays

En este día aprenderás la desestructuración de arrays.

## Contenido
- Sintaxis de destructuring
- Valores por defecto
- Rest operator con arrays
- Intercambiar variables

## Tareas

### Tarea 1: Destructuring básico
Practica:
- Extraer los primeros elementos de un array
- Saltar elementos con comas
- Asignar valores por defecto

### Tarea 2: Rest operator
Practica:
- Obtener el resto de elementos con ...rest
- Combinar destructuring con rest
- Uso práctico en funciones

### Tarea 3: Intercambiar variables
Usa destructuring para:
- Intercambiar el valor de dos variables
- Intercambiar elementos de un array

## Ejercicios Adicionales
- Experimenta con destructuring anidado
- Practica combinando destructuring con parámetros de funciones
