# Día 2: Métodos de Arrays

En este día aprenderás los métodos más comunes para manipular arrays.

## Contenido
- push, pop (final del array)
- shift, unshift (inicio del array)
- splice, slice
- concat, includes, indexOf

## Tareas

### Tarea 1: Agregar y eliminar elementos
Crea un array y practica:
- Agregar elementos al final (push)
- Eliminar el último elemento (pop)
- Agregar elementos al inicio (unshift)
- Eliminar el primer elemento (shift)

### Tarea 2: Splice y slice
Practica:
- Eliminar elementos del medio con splice
- Insertar elementos con splice
- Copiar una porción del array con slice

### Tarea 3: Buscar y combinar
Practica:
- Verificar si un elemento existe (includes)
- Encontrar el índice de un elemento (indexOf)
- Combinar dos arrays (concat)

## Ejercicios Adicionales
- Experimenta con flat() para aplanar arrays anidados
- Practica con join() para convertir a string
