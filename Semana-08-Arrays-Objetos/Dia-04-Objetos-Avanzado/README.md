# Día 4: Objetos Avanzado

En este día aprenderás conceptos avanzados de objetos.

## Contenido
- this en objetos
- Métodos de objetos
- Shorthand properties
- Computed property names

## Tareas

### Tarea 1: Usar this
Crea un objeto con métodos que usen this:
- Un método que calcule la edad en días
- Un método que retorne el nombre completo
- Un método que modifique una propiedad

### Tarea 2: Shorthand properties
Refactoriza código para usar:
- Shorthand property names (cuando variable = propiedad)
- Shorthand methods (sin function keyword)

### Tarea 3: Computed property names
Crea objetos usando:
- Variables como nombres de propiedades
- Expresiones como nombres de propiedades

## Ejercicios Adicionales
- Experimenta con Object.assign() para copiar objetos
- Practica con el spread operator en objetos {...obj}
