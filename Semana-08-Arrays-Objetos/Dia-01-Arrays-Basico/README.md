# Día 1: Arrays Básico

En este día aprenderás los fundamentos de los arrays en JavaScript.

## Contenido
- Qué es un array
- Crear arrays
- Acceder a elementos
- Modificar elementos

## Tareas

### Tarea 1: Crear arrays
Crea arrays de:
- Números del 1 al 10
- Nombres de frutas
- Datos mixtos (números, strings, booleanos)

### Tarea 2: Acceder a elementos
Practica:
- Acceder al primer y último elemento
- Acceder a elementos por índice
- Obtener la longitud del array

### Tarea 3: Modificar arrays
Practica:
- Cambiar el valor de un elemento específico
- Agregar elementos al final con índice
- Crear un array vacío y llenarlo

## Ejercicios Adicionales
- Experimenta con arrays anidados (matrices)
- Practica con el método Array.isArray()
