# Día 6: Destructuring de Objetos

En este día aprenderás la desestructuración de objetos.

## Contenido
- Sintaxis de destructuring de objetos
- Renombrar propiedades
- Valores por defecto
- Rest operator con objetos

## Tareas

### Tarea 1: Destructuring básico
Practica:
- Extraer propiedades de un objeto
- Renombrar variables al extraer
- Asignar valores por defecto

### Tarea 2: Destructuring anidado
Practica con objetos anidados:
- Extraer propiedades de objetos internos
- Combinar con renombramiento
- Manejar propiedades que pueden no existir

### Tarea 3: Destructuring en funciones
Usa destructuring en parámetros:
- Recibir un objeto y extraer propiedades
- Combinar con valores por defecto
- Rest operator para propiedades adicionales

## Ejercicios Adicionales
- Experimenta con spread operator {...obj, nuevaProp: valor}
- Practica fusionando objetos con spread
