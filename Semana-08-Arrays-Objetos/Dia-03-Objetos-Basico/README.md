# Día 3: Objetos Básico

En este día aprenderás los fundamentos de los objetos en JavaScript.

## Contenido
- Crear objetos
- Propiedades y métodos
- Acceso con notación punto
- Acceso con notación corchetes

## Tareas

### Tarea 1: Crear objetos
Crea un objeto persona con:
- Nombre, edad, ciudad
- Un método que imprima una presentación

### Tarea 2: Acceder a propiedades
Practica:
- Acceder con notación punto (persona.nombre)
- Acceder con corchetes (persona["nombre"])
- Acceder a propiedades dinámicamente

### Tarea 3: Modificar objetos
Practica:
- Cambiar valores de propiedades existentes
- Agregar nuevas propiedades
- Eliminar propiedades con delete

## Ejercicios Adicionales
- Experimenta con Object.keys() y Object.values()
- Practica con Object.entries()
