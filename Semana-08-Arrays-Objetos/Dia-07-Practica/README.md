## Proyecto del Día

### Proyecto: Sistema de Gestión de Estudiantes

Crea un sistema que maneje una lista de estudiantes:

1. **Estructura de datos:**
   ```javascript
   const estudiante = {
     id: 1,
     nombre: "Juan",
     apellido: "García",
     calificaciones: [85, 90, 78, 92],
     curso: "Desarrollo Web"
   };
   ```

2. **Funciones requeridas:**
   - `agregarEstudiante(lista, estudiante)` - agrega usando spread
   - `eliminarEstudiante(lista, id)` - usa filter
   - `buscarEstudiante(lista, id)` - usa find
   - `calcularPromedio(estudiante)` - calcula promedio de calificaciones
   - `obtenerMejoresEstudiantes(lista, n)` - retorna los n mejores

3. **Usa destructuring:**
   - En parámetros de funciones
   - Para mostrar información
   - Con rest operator

4. **Operaciones adicionales:**
   - Agregar calificación a un estudiante
   - Actualizar información de estudiante
   - Listar todos los estudiantes

## Tareas Adicionales

### Tarea 1: Exportar datos
Crea una función que convierta la lista a formato JSON.

### Tarea 2: Estadísticas
Calcula estadísticas globales: promedio general, mejor y peor estudiante.
