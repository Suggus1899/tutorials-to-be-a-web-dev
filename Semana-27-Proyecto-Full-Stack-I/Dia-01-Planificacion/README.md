# Día 1: Planificación del Proyecto

En este día planificarás tu proyecto full stack.

## Contenido
- Definir requisitos
- User stories
- Diseño de UI/UX
- Planificación de sprints

## Tareas

### Tarea 1: Definir el proyecto
Elige un proyecto:
- Todo app completa
- Blog con autores
- E-commerce simple
- Red social básica

### Tarea 2: User stories
Escribe historias de usuario:
- Como usuario quiero poder registrarme
- Como usuario quiero crear publicaciones
- Como admin quiero gestionar usuarios

### Tarea 3: Planificar tareas
Divide el trabajo:
- Backend: autenticación, API, base de datos
- Frontend: componentes, rutas, estado
- Prioriza las tareas esenciales

## Ejercicios Adicionales
- Crea wireframes básicos
- Define el MVP (Minimum Viable Product)
