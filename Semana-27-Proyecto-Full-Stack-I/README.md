# Semana 27: Proyecto Full Stack - Parte I

## Objetivos de la Semana

Esta semana comenzarás a construir un proyecto completo integrando todo lo aprendido:

### 1. Planificación del proyecto
- Definir requisitos
- Diseñar la arquitectura
- Configurar el entorno

### 2. Backend API
- Configurar Express y base de datos
- Implementar endpoints CRUD
- Autenticación y autorización

### 3. Testing y documentación
- Tests del backend
- Documentación de la API
- Preparar para deployment

## Estructura de la Semana

- **Día 1-2**: Planificación y diseño
- **Día 3-5**: Desarrollo del backend
- **Día 6-7**: Testing y documentación
