# Día 2: Diseño y Arquitectura

En este día diseñarás la arquitectura del proyecto.

## Contenido
- Diagrama de arquitectura
- Diseño de base de datos
- Diseño de API
- Estructura de carpetas

## Tareas

### Tarea 1: Arquitectura
Diseña la arquitectura:
- Frontend: React + Router + Redux/Context
- Backend: Express + Prisma + SQLite
- API REST entre ambos

### Tarea 2: Diseño de BD
Crea el schema:
- Identifica entidades
- Define relaciones
- Crea diagrama ER
- Escribe Prisma schema

### Tarea 3: Diseño de API
Documenta los endpoints:
- Auth: /api/auth/register, login
- Recursos: CRUD para cada entidad
- Define request/response

## Ejercicios Adicionales
- Configura el monorepo o repos separados
- Establece convenciones de código
