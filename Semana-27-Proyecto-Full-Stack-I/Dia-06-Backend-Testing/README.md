# Día 6: Backend - Testing y API

En este día completarás y testearás la API.

## Contenido
- Endpoints CRUD
- Tests de API
- Documentación
- Validación

## Tareas

### Tarea 1: Completar CRUD
Implementa endpoints para todos los recursos:
- GET /api/[resource] - listar
- GET /api/[resource]/:id - obtener uno
- POST /api/[resource] - crear
- PUT /api/[resource]/:id - actualizar
- DELETE /api/[resource]/:id - eliminar

### Tarea 2: Tests de API
Crea tests con Jest y Supertest:
- Test de autenticación
- Tests de CRUD
- Tests de errores

### Tarea 3: Documentación
Documenta la API:
- README con endpoints
- Ejemplos de request/response
- Postman collection (opcional)

## Ejercicios Adicionales
- Agrega paginación
- Implementa filtros y búsqueda
