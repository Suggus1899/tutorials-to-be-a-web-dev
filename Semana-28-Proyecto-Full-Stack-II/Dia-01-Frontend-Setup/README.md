# Día 1: Frontend Setup

En este día configurarás el frontend de React.

## Contenido
- Crear proyecto con Vite
- Estructura de carpetas
- Configurar routing
- Instalar dependencias

## Tareas

### Tarea 1: Crear proyecto
Inicializa el frontend:
```bash
npm create vite@latest frontend -- --template react
cd frontend
npm install
npm install react-router-dom axios
```

### Tarea 2: Estructura de carpetas
Organiza el proyecto:
```
frontend/
  src/
    components/
    pages/
    hooks/
    services/
    context/ o store/
    utils/
```

### Tarea 3: Configurar Router
Setup inicial de rutas:
- Página de inicio
- Login y registro
- Rutas protegidas base

## Ejercicios Adicionales
- Configura alias de paths
- Agrega estilos base (CSS o Tailwind)
