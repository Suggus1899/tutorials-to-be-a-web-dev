# Día 6: Deployment

En este día desplegarás tu aplicación.

## Contenido
- Deploy del backend
- Deploy del frontend
- Variables de entorno
- Dominio y HTTPS

## Tareas

### Tarea 1: Deploy Backend
Despliega en Railway/Render:
- Conecta el repo
- Configura variables de entorno
- Configura la base de datos
- Obtén la URL del backend

### Tarea 2: Deploy Frontend
Despliega en Vercel/Netlify:
- Conecta el repo
- Configura build command
- Configura variable de API URL
- Obtén la URL del frontend

### Tarea 3: Configuración final
Verifica:
- CORS configurado correctamente
- Variables de entorno en producción
- HTTPS funcionando
- Todas las features funcionan

## Ejercicios Adicionales
- Configura dominio personalizado
- Implementa CI/CD completo
