# Día 2: Frontend - Componentes

En este día crearás los componentes principales.

## Contenido
- Layout components
- Componentes de UI
- Formularios
- Componentes de página

## Tareas

### Tarea 1: Layout
Crea componentes de layout:
- Header con navegación
- Footer
- Layout principal
- Sidebar (si aplica)

### Tarea 2: Componentes reutilizables
Crea componentes de UI:
- Button
- Input
- Card
- Modal
- Loading spinner

### Tarea 3: Páginas
Crea las páginas principales:
- Home
- Login
- Register
- Dashboard
- Detalles de recursos

## Ejercicios Adicionales
- Agrega estilos responsive
- Implementa dark mode
