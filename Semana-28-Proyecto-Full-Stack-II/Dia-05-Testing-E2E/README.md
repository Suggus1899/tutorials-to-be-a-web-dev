# Día 5: Testing E2E

En este día implementarás testing end-to-end.

## Contenido
- Setup de Cypress/Playwright
- Tests de flujos críticos
- CI integration
- Mejores prácticas

## Tareas

### Tarea 1: Setup
Configura herramienta de E2E:
```bash
npm install -D cypress
# o
npm install -D @playwright/test
```

### Tarea 2: Tests críticos
Crea tests para:
- Registro de usuario
- Login/logout
- Crear recurso
- Editar y eliminar

### Tarea 3: CI
Agrega tests a CI:
- Inicia backend y frontend
- Ejecuta tests E2E
- Reporta resultados

## Ejercicios Adicionales
- Agrega tests de accesibilidad
- Implementa visual testing
