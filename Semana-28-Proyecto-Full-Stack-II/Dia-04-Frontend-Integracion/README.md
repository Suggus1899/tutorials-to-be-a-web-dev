# Día 4: Frontend - Integración

En este día integrarás frontend con backend.

## Contenido
- Conectar con API
- Flujo de autenticación
- CRUD completo
- Manejo de errores

## Tareas

### Tarea 1: Autenticación completa
Implementa el flujo:
- Formulario de registro → API → Guardar token
- Formulario de login → API → Guardar token
- Logout → Limpiar token

### Tarea 2: CRUD de recursos
Implementa:
- Listar recursos desde API
- Crear nuevo recurso
- Editar recurso
- Eliminar recurso

### Tarea 3: UX
Mejora la experiencia:
- Loading states
- Mensajes de éxito/error
- Confirmaciones
- Optimistic updates

## Ejercicios Adicionales
- Implementa paginación
- Agrega búsqueda y filtros
