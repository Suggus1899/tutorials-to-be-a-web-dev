# Día 1: Selectores Básicos

En este día aprenderás los selectores CSS básicos.

## Contenido
- Selector de elemento
- Selector de clase
- Selector de ID
- Selectores de atributo

## Tareas

### Tarea 1: Selectores de elemento
Crea un archivo CSS que estilice elementos HTML básicos como h1, p, a, img.

### Tarea 2: Selectores de clase
Crea clases CSS reutilizables para:
- Botones (btn, btn-primary, btn-secondary)
- Textos (text-center, text-bold)
- Contenedores (container, card)

### Tarea 3: Selectores de ID
Usa selectores de ID para elementos únicos como el header o el footer de tu página.

## Ejercicios Adicionales
- Experimenta con selectores de atributo [type="text"]
- Practica combinando múltiples selectores
