# Día 2: Especificidad

En este día aprenderás sobre especificidad y cascada en CSS.

## Contenido
- Cálculo de especificidad
- Orden de la cascada
- !important y por qué evitarlo
- Herencia en CSS

## Tareas

### Tarea 1: Entender la especificidad
Crea ejemplos donde diferentes selectores compitan y observa cuál gana:
- elemento vs clase
- clase vs ID
- ID vs !important

### Tarea 2: Calcular especificidad
Dado los siguientes selectores, ordénalos por especificidad:
- `#header .nav a`
- `.nav-link`
- `header nav a.nav-link`
- `a`

### Tarea 3: Resolver conflictos
Crea un archivo CSS con conflictos de estilos y resuélvelos sin usar !important.

## Ejercicios Adicionales
- Usa las DevTools para inspeccionar qué estilos se aplican
- Practica organizando tu CSS para minimizar conflictos
