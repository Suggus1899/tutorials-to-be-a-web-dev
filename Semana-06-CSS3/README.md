# Semana 6: CSS3

## Objetivos de la Semana

Esta semana aprenderás a estilizar páginas web con CSS3:

### 1. Selectores
- Selectores básicos
- Selectores de clase e ID
- Selectores complejos y pseudo-clases

### 2. Box Model
- Margin, border, padding, content
- width y height
- box-sizing

### 3. Posicionamiento
- static, relative, absolute, fixed
- z-index
- display (block, inline, inline-block)

## Estructura de la Semana

- **Día 1-2**: Selectores y especificidad
- **Día 3-4**: Box model y dimensiones
- **Día 5-6**: Posicionamiento y display
- **Día 7**: Práctica integradora
