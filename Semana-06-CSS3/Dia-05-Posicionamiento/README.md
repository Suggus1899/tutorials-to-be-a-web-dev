# Día 5: Posicionamiento

En este día aprenderás los diferentes tipos de posicionamiento en CSS.

## Contenido
- position: static (default)
- position: relative
- position: absolute
- position: fixed
- position: sticky

## Tareas

### Tarea 1: Position relative
Crea elementos y desplázalos usando position: relative con top, left, right, bottom.

### Tarea 2: Position absolute
Crea un contenedor con position: relative y elementos hijos con position: absolute.
- Coloca un badge en la esquina de una card
- Centra un elemento absolutamente

### Tarea 3: Position fixed
Crea:
- Un header fijo en la parte superior
- Un botón "scroll to top" fijo en la esquina inferior

## Ejercicios Adicionales
- Experimenta con position: sticky para crear un menú sticky
- Practica con z-index para controlar el apilamiento
