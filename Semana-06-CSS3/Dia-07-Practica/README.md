## Proyecto del Día

### Proyecto: Landing Page Estilizada

Crea una landing page con estilos CSS que incluya:

1. **Header:**
   - Position fixed
   - Navegación con inline-block
   - Logo a la izquierda

2. **Hero Section:**
   - Altura de 100vh
   - Contenido centrado
   - Texto con estilos llamativos

3. **Sección de características:**
   - Cards usando box model
   - Distribuidas con inline-block o float
   - Hover effects

4. **Footer:**
   - Fondo oscuro
   - Links con estilos personalizados
   - Información de copyright

5. **Estilos globales:**
   - Reset CSS básico
   - Variables CSS para colores
   - Tipografía consistente

## Tareas Adicionales

### Tarea 1: Agregar transiciones
Añade transiciones suaves a los hover effects.

Importa y usa fuentes de Google Fonts.
