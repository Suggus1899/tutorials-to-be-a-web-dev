# Día 6: Display

En este día aprenderás las diferentes propiedades de display.

## Contenido
- display: block
- display: inline
- display: inline-block
- display: none vs visibility: hidden

## Tareas

### Tarea 1: Diferencia entre block e inline
Crea ejemplos que demuestren:
- Cómo los elementos block ocupan todo el ancho
- Cómo los elementos inline fluyen en línea
- Qué propiedades se pueden aplicar a cada uno

### Tarea 2: Inline-block
Crea una navegación horizontal usando inline-block:
- Elementos que fluyan en línea
- Pero que acepten width, height, margin, padding

### Tarea 3: Ocultar elementos
Demuestra la diferencia entre:
- display: none (elimina del flujo)
- visibility: hidden (oculta pero mantiene espacio)
- opacity: 0 (invisible pero interactivo)

## Ejercicios Adicionales
- Crea un menú de navegación con inline-block
- Experimenta con display: contents
