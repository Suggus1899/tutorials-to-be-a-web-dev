# Día 3: Box Model

En este día aprenderás el modelo de caja de CSS.

## Contenido
- Content, padding, border, margin
- Box-sizing: content-box vs border-box
- Visualización del box model en DevTools
- Colapsado de márgenes

## Tareas

### Tarea 1: Experimentar con el box model
Crea una div y experimenta:
- Agrega padding y observa cómo crece
- Agrega border y observa el efecto
- Agrega margin y ve el espacio exterior

### Tarea 2: Box-sizing
Compara el comportamiento de un elemento con:
- `box-sizing: content-box` (default)
- `box-sizing: border-box`

### Tarea 3: Crear cards
Crea tarjetas (cards) aplicando correctamente:
- Padding interno
- Border redondeado
- Margin entre cards
- Shadow para efecto de elevación

## Ejercicios Adicionales
- Investiga el colapsado de márgenes verticales
- Practica usando shorthand properties (margin: 10px 20px)
