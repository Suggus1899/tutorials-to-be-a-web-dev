# Día 4: Dimensiones

En este día aprenderás a controlar las dimensiones de los elementos.

## Contenido
- Width y height
- Min-width, max-width
- Min-height, max-height
- Unidades de medida (px, %, em, rem, vw, vh)

## Tareas

### Tarea 1: Unidades absolutas vs relativas
Crea elementos usando diferentes unidades:
- Elementos con ancho en px
- Elementos con ancho en %
- Elementos con ancho en vw

### Tarea 2: Min y max width
Crea un contenedor que:
- Tenga un ancho mínimo de 300px
- Tenga un ancho máximo de 1200px
- Se adapte entre esos valores

### Tarea 3: Responsive sin media queries
Usa min-width, max-width y % para crear un layout que se adapte al tamaño de la ventana.

## Ejercicios Adicionales
- Experimenta con unidades em y rem
- Practica con viewport units (vw, vh, vmin, vmax)
