# Día 1: Introducción a JWT

En este día aprenderás qué es JWT y cómo funciona.

## Contenido
- Qué es JWT (JSON Web Tokens)
- Estructura de un token
- Casos de uso
- Ventajas y desventajas

## Tareas

### Tarea 1: Estructura de JWT
Estudia las partes de un JWT:
- Header (algoritmo y tipo)
- Payload (claims/datos)
- Signature (firma)

### Tarea 2: Analizar tokens
Usa jwt.io para:
- Decodificar tokens de ejemplo
- Ver las partes del token
- Entender la codificación Base64

### Tarea 3: Conceptos clave
Documenta:
- Diferencia entre autenticación y autorización
- Stateless vs stateful authentication
- Access tokens vs refresh tokens

## Ejercicios Adicionales
- Investiga otros métodos de autenticación (sessions, OAuth)
- Compara JWT con cookies de sesión
