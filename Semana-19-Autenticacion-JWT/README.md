# Semana 19: Autenticación con JWT

## Objetivos de la Semana

Esta semana aprenderás a implementar autenticación y autorización:

### 1. Introducción a JWT
- Qué es JWT (JSON Web Tokens)
- Estructura de un token JWT
- Ventajas y desventajas

### 2. Implementación de autenticación
- Registro de usuarios (hash de contraseñas)
- Login y generación de tokens
- Middleware de autenticación

### 3. Autorización y buenas prácticas
- Roles y permisos
- Refresh tokens
- Seguridad en JWT

## Estructura de la Semana

- **Día 1-2**: Introducción a JWT y conceptos de seguridad
- **Día 3-4**: Implementar registro y login
- **Día 5-6**: Middleware y autorización
- **Día 7**: Práctica integradora
