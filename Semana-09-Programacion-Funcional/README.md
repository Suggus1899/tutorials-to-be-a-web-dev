# Semana 9: Programación Funcional

## Objetivos de la Semana

Esta semana aprenderás los fundamentos de programación funcional:

### 1. map()
- Transformar arrays
- Sintaxis y uso
- Casos prácticos

### 2. filter()
- Filtrar elementos
- Condiciones de filtrado
- Combinar con otras operaciones

### 3. reduce()
- Reducir arrays a un valor
- Acumuladores
- Casos de uso avanzados

## Estructura de la Semana

- **Día 1-2**: map() y transformaciones
- **Día 3-4**: filter() y filtrado de datos
- **Día 5-6**: reduce() y agregaciones
- **Día 7**: Práctica integradora
