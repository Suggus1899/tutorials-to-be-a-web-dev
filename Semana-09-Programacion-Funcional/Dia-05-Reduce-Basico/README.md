# Día 5: Reduce Básico

En este día aprenderás los fundamentos del método reduce().

## Contenido
- Qué es reduce()
- Acumulador y valor actual
- Valor inicial
- Casos de uso comunes

## Tareas

### Tarea 1: Operaciones numéricas
Usa reduce() para:
- Sumar todos los elementos de un array
- Encontrar el valor máximo
- Encontrar el valor mínimo
- Calcular el promedio

### Tarea 2: Reducir a otros tipos
Practica reduciendo a:
- Un string (concatenar elementos)
- Un boolean (verificar si todos cumplen condición)
- Un número (contar elementos)

### Tarea 3: Sumar propiedades de objetos
Dado un array de productos:
- Suma el precio total
- Suma las cantidades totales
- Calcula el valor del inventario

## Ejercicios Adicionales
- Implementa tu propia versión de sum() con reduce
- Practica con diferentes valores iniciales
