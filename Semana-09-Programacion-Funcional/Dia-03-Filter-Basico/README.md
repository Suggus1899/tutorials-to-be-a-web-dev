# Día 3: Filter Básico

En este día aprenderás los fundamentos del método filter().

## Contenido
- Qué es filter()
- Sintaxis básica
- Funciones predicado
- Filtrado vs búsqueda

## Tareas

### Tarea 1: Filtros simples
Usa filter() para:
- Obtener solo números pares
- Filtrar strings que empiecen con una letra
- Eliminar valores falsy (null, undefined, 0, "")

### Tarea 2: Filtrar objetos
Dado un array de productos:
- Filtra productos con precio mayor a 50
- Filtra productos en stock (cantidad > 0)
- Filtra por categoría

### Tarea 3: Múltiples condiciones
Practica con filtros complejos:
- Productos baratos Y en stock
- Personas mayores de edad O empleados
- Combinar && y || en condiciones

## Ejercicios Adicionales
- Compara filter() con find()
- Experimenta con filter() y includes()
