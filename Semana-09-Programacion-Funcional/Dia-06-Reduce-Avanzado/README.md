# Día 6: Reduce Avanzado

En este día aprenderás usos avanzados de reduce().

## Contenido
- Reducir a objetos
- Agrupar elementos
- Aplanar arrays
- Implementar otros métodos con reduce

## Tareas

### Tarea 1: Agrupar elementos
Usa reduce() para:
- Agrupar productos por categoría
- Contar ocurrencias de cada elemento
- Crear un índice por ID

### Tarea 2: Transformar estructuras
Practica:
- Convertir array a objeto
- Aplanar arrays anidados
- Crear objetos lookup

### Tarea 3: Reimplementar métodos
Usa reduce() para implementar:
- map() usando reduce
- filter() usando reduce
- find() usando reduce

## Ejercicios Adicionales
- Implementa pipe() con reduce para composición de funciones
- Practica creando reducers para estado (preview de Redux)
