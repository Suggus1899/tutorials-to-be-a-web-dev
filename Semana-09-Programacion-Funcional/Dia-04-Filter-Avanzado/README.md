# Día 4: Filter Avanzado

En este día aprenderás usos avanzados de filter().

## Contenido
- Filtros dinámicos
- Combinar filter con map
- Eliminar duplicados
- Filtrado en datos anidados

## Tareas

### Tarea 1: Filtros dinámicos
Crea funciones que:
- Reciban el criterio de filtrado como parámetro
- Filtren por cualquier propiedad
- Construyan filtros dinámicamente

### Tarea 2: Combinar filter y map
Practica encadenando métodos:
- filter().map() para filtrar y transformar
- Obtener solo ciertos datos de elementos filtrados
- Crear pipelines de datos

### Tarea 3: Casos especiales
Practica con:
- Eliminar duplicados usando filter e indexOf
- Filtrar objetos anidados
- Intersección de dos arrays

## Ejercicios Adicionales
- Implementa una función de búsqueda con múltiples criterios
- Practica con every() y some() para validaciones
