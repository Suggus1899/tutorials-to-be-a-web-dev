# Día 1: Map Básico

En este día aprenderás los fundamentos del método map().

## Contenido
- Qué es map()
- Sintaxis básica
- Transformar arrays
- Diferencia con forEach

## Tareas

### Tarea 1: Transformaciones simples
Usa map() para:
- Duplicar todos los números de un array
- Convertir strings a mayúsculas
- Agregar un prefijo a cada elemento

### Tarea 2: Extraer propiedades
Dado un array de objetos personas:
- Extrae solo los nombres
- Extrae solo las edades
- Crea strings de presentación

### Tarea 3: Map con índice
Practica usando el segundo parámetro (índice):
- Numerar elementos
- Crear objetos con posición
- Operaciones que dependen del índice

## Ejercicios Adicionales
- Compara map() con un ciclo for tradicional
- Experimenta con transformaciones complejas
