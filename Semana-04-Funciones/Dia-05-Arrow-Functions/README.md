# Día 5: Arrow Functions

En este día aprenderás la sintaxis de arrow functions.

## Contenido
- Sintaxis de arrow functions
- Retorno implícito
- Diferencias con funciones tradicionales
- Contexto de `this` en arrow functions

## Tareas

### Tarea 1: Convertir a arrow functions
Toma las funciones que creaste en días anteriores y conviértelas a arrow functions.

### Tarea 2: Retorno implícito
Practica escribiendo arrow functions de una sola línea con retorno implícito.

### Tarea 3: Arrow functions con métodos de array
Usa arrow functions con map, filter y forEach en un array de números.

## Ejercicios Adicionales
- Investiga las diferencias de `this` entre funciones tradicionales y arrow functions
- Practica combinando arrow functions con destructuración
