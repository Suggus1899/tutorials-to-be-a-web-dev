## Proyecto del Día

### Proyecto: Biblioteca de Utilidades Matemáticas

Crea una biblioteca de funciones matemáticas que incluya:

1. **Funciones básicas:**
   - `sumar(a, b)`
   - `restar(a, b)`
   - `multiplicar(a, b)`
   - `dividir(a, b)` (con validación de división por cero)

2. **Funciones intermedias:**
   - `calcularPromedio(...numeros)` - usando rest parameters
   - `calcularPotencia(base, exponente = 2)` - con parámetro por defecto
   - `esParOImpar(numero)` - retorna "par" o "impar"

3. **Funciones avanzadas:**
   - `crearCalculadora()` - retorna un objeto con métodos de calculadora
   - `factorial(n)` - implementa el factorial usando recursión

4. **Documentación:**
   - Documenta cada función con JSDoc

## Tareas Adicionales
