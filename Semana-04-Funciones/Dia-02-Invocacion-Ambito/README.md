# Día 2: Invocación y Ámbito

En este día aprenderás sobre invocación de funciones y el ámbito de variables.

## Contenido
- Cómo invocar funciones
- Ámbito local vs global
- Closure básico
- Contexto de ejecución

## Tareas

### Tarea 1: Variables locales y globales
Crea un ejemplo que demuestre la diferencia entre variables locales y globales.

### Tarea 2: Ámbito anidado
Crea funciones anidadas y observa cómo las funciones internas pueden acceder a variables de funciones externas.

### Tarea 3: Introducción a closures
Crea una función que devuelva otra función y demuestre el concepto de closure.

## Ejercicios Adicionales
- Experimenta con el shadowing de variables
- Investiga qué es el lexical scope
