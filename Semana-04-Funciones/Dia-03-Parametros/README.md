# Día 3: Parámetros

En este día aprenderás a trabajar con parámetros en funciones.

## Contenido
- Parámetros y argumentos
- Parámetros por defecto
- Rest parameters (...args)
- Validación de parámetros

## Tareas

### Tarea 1: Función con múltiples parámetros
Crea una función `presentarse` que reciba nombre, edad y ciudad, y retorne una presentación completa.

### Tarea 2: Parámetros por defecto
Modifica la función anterior para que tenga valores por defecto para cada parámetro.

### Tarea 3: Rest parameters
Crea una función `sumarTodos` que reciba cualquier cantidad de números y retorne su suma.

## Ejercicios Adicionales
- Practica la validación de parámetros dentro de funciones
- Experimenta con la destructuración de parámetros
