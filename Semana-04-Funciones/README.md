# Semana 4: Funciones

## Objetivos de la Semana

Esta semana aprenderás a crear y utilizar funciones:

### 1. Fundamentos de funciones
- Declaración de funciones
- Invocación de funciones
- Ámbito de las funciones

### 2. Parámetros y return
- Parámetros y argumentos
- Valores de retorno
- Parámetros por defecto

### 3. Arrow functions
- Sintaxis de arrow functions
- Diferencias con funciones tradicionales
- Cuándo usar cada tipo

## Estructura de la Semana

- **Día 1-2**: Declaración y uso de funciones
- **Día 3-4**: Parámetros y valores de retorno
- **Día 5-6**: Arrow functions y mejores prácticas
- **Día 7**: Práctica integradora
