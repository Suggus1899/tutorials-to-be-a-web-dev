# Día 6: Mejores Prácticas

En este día aprenderás las mejores prácticas para escribir funciones.

## Contenido
- Funciones puras
- Principio de responsabilidad única
- Nombres descriptivos
- Documentación de funciones

## Tareas

### Tarea 1: Refactorizar a funciones puras
Toma funciones que modifiquen estado externo y refactorízalas para que sean puras.

### Tarea 2: Dividir funciones grandes
Si tienes una función que hace múltiples cosas, divídela en funciones más pequeñas.

### Tarea 3: Documentar funciones
Agrega comentarios JSDoc a tus funciones indicando parámetros y valores de retorno.

## Ejercicios Adicionales
- Revisa código anterior y aplica mejores prácticas
- Investiga sobre composición de funciones
