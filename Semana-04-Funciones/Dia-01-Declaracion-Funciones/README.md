# Día 1: Declaración de Funciones

En este día aprenderás cómo declarar funciones en JavaScript.

## Contenido
- Sintaxis de declaración de funciones
- Function declaration vs function expression
- Hoisting en funciones
- Nombres descriptivos

## Tareas

### Tarea 1: Crear tu primera función
Crea una función llamada `saludar` que imprima "¡Hola, mundo!" en la consola.

### Tarea 2: Function expression
Reescribe la función `saludar` como una function expression asignada a una variable.

### Tarea 3: Múltiples funciones
Crea tres funciones: `sumar`, `restar` y `multiplicar` que realicen operaciones matemáticas básicas.

## Ejercicios Adicionales
- Experimenta con hoisting llamando funciones antes de declararlas
- Practica nombrando funciones de forma descriptiva
