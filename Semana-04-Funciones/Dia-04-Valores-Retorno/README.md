# Día 4: Valores de Retorno

En este día aprenderás sobre return y valores de retorno.

## Contenido
- La palabra clave return
- Retornar diferentes tipos de datos
- Retornar múltiples valores
- Early return pattern

## Tareas

### Tarea 1: Funciones que retornan valores
Crea una función `calcularAreaRectangulo` que reciba base y altura, y retorne el área.

### Tarea 2: Retornar objetos
Crea una función que retorne un objeto con múltiples propiedades calculadas.

### Tarea 3: Early return
Crea una función que valide datos usando early return para manejar casos de error.

## Ejercicios Adicionales
- Practica con funciones que retornan arrays
- Experimenta con funciones que retornan otras funciones
