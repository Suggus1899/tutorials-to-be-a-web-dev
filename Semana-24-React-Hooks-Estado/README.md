# Semana 24: React Hooks y Estado

## Objetivos de la Semana

Esta semana aprenderás a manejar el estado en React:

### 1. useState Hook
- Estado en componentes funcionales
- Actualizar estado
- Estado con objetos y arrays

### 2. useEffect Hook
- Efectos secundarios
- Ciclo de vida del componente
- Cleanup y dependencias

### 3. Hooks adicionales
- useRef y DOM refs
- useMemo y useCallback
- Custom hooks

## Estructura de la Semana

- **Día 1-2**: useState y manejo de estado
- **Día 3-4**: useEffect y ciclo de vida
- **Día 5-6**: Hooks avanzados y custom hooks
- **Día 7**: Práctica integradora
