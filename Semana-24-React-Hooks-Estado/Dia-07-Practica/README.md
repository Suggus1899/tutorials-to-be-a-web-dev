## Proyecto del Día

### Proyecto: Aplicación de Notas con Persistencia

Crea una aplicación de notas:

1. **Estado con useState:**
   - Lista de notas
   - Nota en edición
   - Filtros activos

2. **useEffect:**
   - Guardar en localStorage
   - Cargar al iniciar
   - Actualizar título del documento

3. **Funcionalidades:**
   - Crear nota
   - Editar nota
   - Eliminar nota
   - Buscar notas
   - Marcar como favorita

4. **Custom hooks:**
   - useLocalStorage para persistencia
   - useNotes con lógica de CRUD
   - useFilter para filtrado

5. **Interfaz:**
   - Lista de notas
   - Formulario de edición
   - Barra de búsqueda
   - Toggle dark mode
