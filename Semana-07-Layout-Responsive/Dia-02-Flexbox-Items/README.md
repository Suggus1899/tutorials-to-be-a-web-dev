# Día 2: Flexbox - Propiedades de Items

En este día aprenderás las propiedades que se aplican a los items flex.

## Contenido
- align-items y align-self
- flex-wrap
- flex-grow, flex-shrink, flex-basis
- Shorthand flex

## Tareas

### Tarea 1: Align-items
Crea un contenedor con elementos de diferente altura y usa:
- align-items: stretch (default)
- align-items: center
- align-items: flex-start / flex-end

### Tarea 2: Flex-wrap
Crea una galería de imágenes que:
- Use flex-wrap: wrap para que los elementos pasen a la siguiente línea
- Se adapte al ancho del contenedor

### Tarea 3: Flex-grow y flex-shrink
Crea un layout donde:
- Un elemento ocupe el espacio restante (flex-grow: 1)
- Un sidebar tenga ancho fijo
- El contenido principal crezca

## Ejercicios Adicionales
- Experimenta con flex-basis
- Practica usando el shorthand: flex: 1 1 0%
