# Día 1: Flexbox Básico

En este día aprenderás los fundamentos de Flexbox.

## Contenido
- display: flex
- Eje principal y eje transversal
- flex-direction
- justify-content

## Tareas

### Tarea 1: Contenedor flex
Crea un contenedor con varios elementos hijos y aplica display: flex. Observa cómo cambian los elementos.

### Tarea 2: Flex-direction
Experimenta con todas las direcciones:
- row (default)
- row-reverse
- column
- column-reverse

### Tarea 3: Justify-content
Crea una barra de navegación y usa justify-content para:
- Centrar elementos (center)
- Espaciar elementos (space-between, space-around, space-evenly)
- Alinear al inicio o fin

## Ejercicios Adicionales
- Crea un menú horizontal con elementos centrados
- Experimenta combinando flex-direction y justify-content
