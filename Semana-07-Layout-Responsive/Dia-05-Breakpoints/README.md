# Día 5: Breakpoints

En este día aprenderás a elegir y usar breakpoints efectivamente.

## Contenido
- Breakpoints comunes
- Estrategia Mobile-first
- Organización de media queries
- Breakpoints personalizados

## Tareas

### Tarea 1: Breakpoints estándar
Implementa una página usando estos breakpoints:
- Mobile: < 576px
- Tablet: 576px - 992px
- Desktop: > 992px

### Tarea 2: Mobile-first
Refactoriza una página existente para que:
- Los estilos base sean para móvil
- Use min-width para agregar estilos en pantallas más grandes
- Sea más eficiente en términos de CSS

### Tarea 3: Contenido adaptativo
Crea una sección que:
- Muestre texto corto en móvil
- Muestre texto completo en desktop
- Cambie el tamaño de fuente según el dispositivo

## Ejercicios Adicionales
- Investiga los breakpoints usados por frameworks como Bootstrap
- Crea tus propias variables CSS para breakpoints
