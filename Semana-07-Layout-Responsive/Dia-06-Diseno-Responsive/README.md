# Día 6: Diseño Responsive Completo

En este día aprenderás técnicas avanzadas de diseño responsive.

## Contenido
- Unidades relativas (%, em, rem, vw, vh)
- Imágenes responsive
- Tipografía fluida
- CSS Grid básico

## Tareas

### Tarea 1: Imágenes responsive
Implementa imágenes que:
- Usen max-width: 100%
- Mantengan su aspect ratio
- Usen el elemento picture para diferentes resoluciones

### Tarea 2: Tipografía fluida
Crea títulos que:
- Escalen con el tamaño de la ventana
- Usen clamp() para límites
- Sean legibles en todos los tamaños

### Tarea 3: Layout completo responsive
Crea una página que:
- Use unidades relativas en todo
- Se vea bien desde 320px hasta 1920px
- No tenga scroll horizontal en ningún tamaño

## Ejercicios Adicionales
- Experimenta con CSS Grid para layouts complejos
- Practica con aspect-ratio para mantener proporciones
