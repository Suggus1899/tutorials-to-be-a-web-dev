# Semana 7: Layout Responsive

## Objetivos de la Semana

Esta semana aprenderás a crear layouts adaptativos:

### 1. Flexbox
- display: flex
- Propiedades del contenedor
- Propiedades de los items

### 2. Media Queries
- Sintaxis de media queries
- Breakpoints comunes
- Mobile-first vs Desktop-first

### 3. Diseño Responsive
- Unidades relativas (%, em, rem, vw, vh)
- Imágenes responsive
- Grids con flexbox

## Estructura de la Semana

- **Día 1-3**: Flexbox y sus propiedades
- **Día 4-5**: Media queries y breakpoints
- **Día 6**: Diseño responsive completo
- **Día 7**: Práctica integradora
