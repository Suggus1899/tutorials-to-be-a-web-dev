# Día 4: Media Queries

En este día aprenderás a usar media queries.

## Contenido
- Sintaxis de media queries
- Breakpoints
- Mobile-first vs Desktop-first
- Combinar condiciones

## Tareas

### Tarea 1: Media query básica
Crea una página donde:
- El fondo sea azul en pantallas pequeñas
- El fondo sea verde en pantallas grandes
- Usa min-width o max-width

### Tarea 2: Navegación responsive
Crea una navegación que:
- Sea horizontal en desktop
- Se convierta en vertical en móvil
- Oculte algunos elementos en móvil

### Tarea 3: Cards responsive
Modifica tu grid de cards para que:
- Muestre 1 card por fila en móvil
- Muestre 2 cards por fila en tablet
- Muestre 3 cards por fila en desktop

## Ejercicios Adicionales
- Experimenta con media queries para orientación
- Practica con print media queries
