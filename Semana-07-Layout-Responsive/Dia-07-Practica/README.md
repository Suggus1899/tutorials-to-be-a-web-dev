## Proyecto del Día

### Proyecto: Sitio Web Responsive Completo

Crea un sitio web completamente responsive que incluya:

1. **Header responsive:**
   - Navegación horizontal en desktop
   - Menú hamburguesa en móvil (solo CSS, sin JS)
   - Logo que se adapte

2. **Hero section:**
   - Imagen de fondo responsive
   - Texto que escale apropiadamente
   - Botones que se reorganicen

3. **Grid de servicios:**
   - 3 columnas en desktop
   - 2 columnas en tablet
   - 1 columna en móvil

4. **Testimonios:**
   - Carousel horizontal en desktop
   - Cards apiladas en móvil

5. **Footer:**
   - Múltiples columnas en desktop
   - Apilado en móvil

## Tareas Adicionales
