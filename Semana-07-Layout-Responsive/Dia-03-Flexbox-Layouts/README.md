# Día 3: Flexbox - Layouts Comunes

En este día crearás layouts comunes usando Flexbox.

## Contenido
- Layout de navegación
- Layout de cards
- Layout de dos columnas
- Centrado perfecto

## Tareas

### Tarea 1: Navegación completa
Crea una barra de navegación con:
- Logo a la izquierda
- Links en el centro
- Botón de login a la derecha

### Tarea 2: Grid de cards
Crea un grid de cards que:
- Muestre 3 cards por fila en pantallas grandes
- Se ajuste con flex-wrap
- Tenga gap entre elementos

### Tarea 3: Layout de dos columnas
Crea un layout con:
- Sidebar de 300px
- Contenido principal que ocupe el resto
- Ambos con altura igual

## Ejercicios Adicionales
- Crea un footer con múltiples columnas
- Implementa el patrón "Holy Grail Layout"
