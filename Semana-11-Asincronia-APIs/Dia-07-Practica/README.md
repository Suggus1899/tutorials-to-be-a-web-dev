## Proyecto del Día

### Proyecto: Buscador de Películas con API

Crea una aplicación que consuma una API de películas:

1. **Usar API de películas:**
   - OMDB API (necesita key gratuita) o
   - TMDB API o
   - Cualquier API de películas pública

2. **Funcionalidades:**
   - Buscar películas por título
   - Mostrar resultados en cards
   - Ver detalles de una película
   - Guardar favoritos en localStorage

3. **Implementación técnica:**
   - Usar async/await para las peticiones
   - Manejar errores (API no disponible, sin resultados)
   - Mostrar estados de carga
   - Debounce en la búsqueda

4. **Interfaz:**
   - Input de búsqueda
   - Grid de resultados
   - Modal o página de detalles
   - Sección de favoritos

## Tareas Adicionales

### Tarea 1: Búsqueda avanzada
Agregar filtros por año, género, etc.

### Tarea 2: Paginación
Implementar carga de más resultados.
