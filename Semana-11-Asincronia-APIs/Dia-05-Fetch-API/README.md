# Día 5: Fetch API

En este día aprenderás a hacer peticiones HTTP con fetch.

## Contenido
- Sintaxis de fetch()
- Trabajar con Response
- Parsear JSON
- Headers y opciones de fetch

## Tareas

### Tarea 1: GET request
Practica:
- Hacer una petición GET a una API pública
- Parsear la respuesta como JSON
- Mostrar los datos en la consola

### Tarea 2: Verificar respuesta
Practica:
- Verificar response.ok
- Manejar diferentes status codes
- Crear mensajes de error descriptivos

### Tarea 3: Opciones de fetch
Practica con:
- Headers personalizados
- Diferentes métodos (GET, POST)
- Enviar datos en el body

## Ejercicios Adicionales
- Experimenta con APIs públicas (JSONPlaceholder, PokeAPI)
- Practica con AbortController para cancelar peticiones
