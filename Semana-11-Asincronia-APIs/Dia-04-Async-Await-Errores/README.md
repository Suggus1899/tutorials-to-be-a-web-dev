# Día 4: Async/Await - Manejo de Errores

En este día aprenderás a manejar errores con async/await.

## Contenido
- try/catch con async/await
- Manejo de múltiples errores
- Crear funciones robustas
- Patrones de manejo de errores

## Tareas

### Tarea 1: try/catch básico
Practica:
- Envolver await en try/catch
- Capturar y manejar errores
- Usar finally para limpieza

### Tarea 2: Errores en operaciones paralelas
Practica:
- Manejar errores con Promise.all
- Continuar aunque una operación falle
- Reportar errores parciales

### Tarea 3: Patrones de error
Implementa:
- Función wrapper para try/catch
- Reintentos automáticos (retry pattern)
- Fallback cuando una operación falla

## Ejercicios Adicionales
- Crea clases de error personalizadas
- Implementa logging de errores
