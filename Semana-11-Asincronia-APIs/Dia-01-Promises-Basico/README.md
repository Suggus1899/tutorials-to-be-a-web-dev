# Día 1: Promises Básico

En este día aprenderás los fundamentos de las Promesas en JavaScript.

## Contenido
- Qué es una Promise
- Estados de una Promise (pending, fulfilled, rejected)
- then() y catch()
- finally()

## Tareas

### Tarea 1: Crear tu primera Promise
Crea una promesa que:
- Se resuelva después de 2 segundos
- Retorne un mensaje de éxito
- Maneja el resultado con .then()

### Tarea 2: Manejar errores
Crea una promesa que:
- Se rechace con un error
- Maneja el error con .catch()
- Usa .finally() para código de limpieza

### Tarea 3: Simular operaciones asíncronas
Crea funciones que simulen:
- Obtener datos de un usuario (con setTimeout)
- Verificar credenciales (éxito o error aleatorio)
- Cargar un archivo (con progreso)

## Ejercicios Adicionales
- Experimenta con Promise.resolve() y Promise.reject()
- Practica encadenando promesas simples
