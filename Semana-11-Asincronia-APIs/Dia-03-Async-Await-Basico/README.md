# Día 3: Async/Await Básico

En este día aprenderás la sintaxis de async/await.

## Contenido
- Funciones async
- Palabra clave await
- Comparación con then/catch
- Cuándo usar cada enfoque

## Tareas

### Tarea 1: Convertir promesas a async/await
Toma los ejercicios del Día 1 y 2 y reescríbelos usando async/await.

### Tarea 2: Funciones async
Crea funciones async que:
- Obtengan datos de forma secuencial
- Esperen un tiempo antes de continuar
- Retornen valores procesados

### Tarea 3: Secuencia vs paralelo
Practica:
- Ejecutar awaits en secuencia (uno tras otro)
- Ejecutar en paralelo con Promise.all y await
- Comparar tiempos de ejecución

## Ejercicios Adicionales
- Experimenta con arrow functions async
- Practica con métodos de objeto async
