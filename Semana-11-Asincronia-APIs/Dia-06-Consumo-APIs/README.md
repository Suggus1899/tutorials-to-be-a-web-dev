# Día 6: Consumo de APIs

En este día aprenderás a consumir APIs REST de forma práctica.

## Contenido
- CRUD con fetch (GET, POST, PUT, DELETE)
- Manejo de respuestas JSON
- Autenticación básica (headers)
- Mejores prácticas

## Tareas

### Tarea 1: Implementar CRUD
Usa JSONPlaceholder o similar para:
- GET: Obtener lista de posts
- POST: Crear un nuevo post
- PUT: Actualizar un post
- DELETE: Eliminar un post

### Tarea 2: Crear servicio API
Crea un módulo/objeto con:
- Funciones para cada operación CRUD
- Manejo de errores centralizado
- URL base configurable

### Tarea 3: Integrar con el DOM
Crea una página que:
- Muestre datos de la API en el DOM
- Permita crear nuevos items
- Permita eliminar items

## Ejercicios Adicionales
- Experimenta con APIs que requieran autenticación
- Implementa paginación de resultados
