# Día 1: Introducción a ORM

En este día aprenderás qué es un ORM y sus beneficios.

## Contenido
- Qué es un ORM
- Ventajas y desventajas
- ORMs populares (Prisma, Sequelize, TypeORM)
- Cuándo usar un ORM

## Tareas

### Tarea 1: Conceptos de ORM
Estudia y documenta:
- Qué significa Object-Relational Mapping
- Cómo un ORM mapea objetos a tablas
- Diferencia entre ORM y query builder

### Tarea 2: Comparar ORMs
Investiga:
- Prisma: moderno, type-safe
- Sequelize: tradicional, feature-rich
- TypeORM: decoradores, TypeScript
- Ventajas de cada uno

### Tarea 3: Evaluar uso de ORM
Considera:
- Cuándo un ORM es útil
- Cuándo SQL raw es mejor
- Trade-offs de abstracción

## Ejercicios Adicionales
- Investiga Active Record vs Data Mapper patterns
- Revisa proyectos que usen diferentes ORMs
