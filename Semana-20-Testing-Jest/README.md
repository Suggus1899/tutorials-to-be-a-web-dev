# Semana 20: Testing con Jest

## Objetivos de la Semana

Esta semana aprenderás a escribir tests para tu código:

### 1. Introducción al testing
- Por qué hacer testing
- Tipos de tests (unitarios, integración, E2E)
- Configuración de Jest

### 2. Tests unitarios
- Escribir test cases
- Matchers y assertions
- Mocking y spies

### 3. Tests de integración
- Testar APIs con supertest
- Tests de base de datos
- Coverage y mejores prácticas

## Estructura de la Semana

- **Día 1-2**: Introducción al testing y configuración de Jest
- **Día 3-4**: Tests unitarios y mocking
- **Día 5-6**: Tests de integración para APIs
- **Día 7**: Práctica integradora
