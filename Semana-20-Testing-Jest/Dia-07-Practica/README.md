## Proyecto del Día

### Proyecto: Suite de Tests Completa

Agrega tests a una API existente:

1. **Tests unitarios:**
   - Testear funciones de utilidad
   - Testear servicios con mocks
   - Testear validaciones
   - Coverage > 80%

2. **Tests de integración:**
   - Testear todos los endpoints
   - Testear flujos completos (registro → login → crear post)
   - Testear casos de error

3. **Mocking:**
   - Mockear base de datos para tests unitarios
   - Mockear servicios externos
   - Usar BD real para integración

4. **Organización:**
   - Carpeta __tests__ o archivos .test.js
   - Separar unitarios de integración
   - Fixtures para datos de prueba
