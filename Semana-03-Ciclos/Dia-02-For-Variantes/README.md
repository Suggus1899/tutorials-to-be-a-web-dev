# Día 2: Variantes del For

En este día aprenderás las variantes modernas del ciclo for.

## Contenido
- for...of para iterar valores
- for...in para iterar propiedades
- forEach para arrays
- Diferencias entre cada variante

## Tareas

### Tarea 1: Iterar un array con for...of
Crea un array de frutas y usa for...of para imprimir cada fruta.

### Tarea 2: Iterar propiedades con for...in
Crea un objeto persona y usa for...in para imprimir todas sus propiedades y valores.

### Tarea 3: Usar forEach
Reescribe la Tarea 1 usando el método forEach.

## Ejercicios Adicionales
- Compara el rendimiento entre for clásico y forEach
- Investiga cuándo es mejor usar cada variante
