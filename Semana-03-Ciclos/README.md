# Semana 3: Ciclos

## Objetivos de la Semana

Esta semana aprenderás a repetir operaciones de manera eficiente:

### 1. Ciclo for
- Sintaxis del for
- for clásico vs for...of
- Iteración sobre arrays

### 2. Ciclo while
- Sintaxis del while
- do...while
- Cuándo usar while vs for

### 3. Control de flujo en ciclos
- break: salir de un ciclo
- continue: saltar a la siguiente iteración
- Evitar ciclos infinitos

## Estructura de la Semana

- **Día 1-2**: Ciclo for y sus variantes
- **Día 3-4**: Ciclo while y do...while
- **Día 5-6**: break, continue y casos prácticos
- **Día 7**: Práctica integradora
