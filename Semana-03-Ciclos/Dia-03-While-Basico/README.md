# Día 3: Ciclo While Básico

En este día aprenderás la sintaxis y uso del ciclo while.

## Contenido
- Sintaxis del while
- Condición de parada
- Evitar ciclos infinitos
- Comparación con for

## Tareas

### Tarea 1: Contador con while
Crea un programa que cuente del 1 al 10 usando while.

### Tarea 2: Adivinar un número
Crea un programa que simule adivinar un número secreto. El ciclo debe continuar hasta que el usuario adivine correctamente.

### Tarea 3: Validación de entrada
Crea un programa que pida al usuario un número positivo y repita la solicitud mientras el número sea negativo o cero.

## Ejercicios Adicionales
- Practica identificando cuándo usar while en lugar de for
- Experimenta con condiciones más complejas
