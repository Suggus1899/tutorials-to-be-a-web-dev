# Día 6: Casos Prácticos

En este día aplicarás ciclos a problemas del mundo real.

## Contenido
- Patrones comunes con ciclos
- Ciclos anidados
- Optimización de ciclos
- Evitar errores comunes

## Tareas

### Tarea 1: Dibujar un triángulo
Crea un programa que dibuje un triángulo de asteriscos:
```
*
**
***
****
*****
```

### Tarea 2: FizzBuzz
Crea el clásico programa FizzBuzz:
- Imprime "Fizz" para múltiplos de 3
- Imprime "Buzz" para múltiplos de 5
- Imprime "FizzBuzz" para múltiplos de ambos
- Imprime el número en otros casos

### Tarea 3: Números primos
Crea un programa que encuentre todos los números primos del 1 al 100.

## Ejercicios Adicionales
- Crea un patrón de cuadrado con asteriscos
- Implementa el algoritmo de la secuencia de Fibonacci
