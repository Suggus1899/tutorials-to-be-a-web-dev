# Día 7: Práctica Integradora

En este día realizarás ejercicios que integran todos los conceptos de la semana.

## Contenido
- Repaso de for y sus variantes
- Repaso de while y do...while
- Repaso de break y continue
- Proyecto integrador

## Proyecto del Día

### Proyecto: Juego de Adivinanza con Estadísticas

Crea un juego que:

1. **Genere un número aleatorio** entre 1 y 100

2. **Permita al usuario adivinar:**
   - Use do...while para garantizar al menos un intento
   - Proporcione pistas ("más alto" o "más bajo")
   - Use break cuando adivine correctamente

3. **Lleve estadísticas:**
   - Número de intentos
   - Historial de números intentados (usar array y for...of)
   - Mejor puntuación guardada

4. **Ofrezca opciones al finalizar:**
   - Jugar de nuevo
   - Ver estadísticas
   - Salir

## Tareas Adicionales

### Tarea 1: Agregar niveles de dificultad
- Fácil: números del 1-10
- Medio: números del 1-50
- Difícil: números del 1-100

### Tarea 2: Límite de intentos
Agrega un límite de intentos según la dificultad.
