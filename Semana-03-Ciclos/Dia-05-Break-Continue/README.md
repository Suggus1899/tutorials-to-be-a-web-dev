# Día 5: Break y Continue

En este día aprenderás a controlar el flujo dentro de los ciclos.

## Contenido
- break: salir de un ciclo
- continue: saltar a la siguiente iteración
- Uso con ciclos anidados
- Mejores prácticas

## Tareas

### Tarea 1: Buscar un elemento
Crea un programa que busque un número específico en un array y use break para salir cuando lo encuentre.

### Tarea 2: Saltar números pares
Crea un programa que imprima solo los números impares del 1 al 20 usando continue.

### Tarea 3: Ciclos anidados con break
Crea una matriz 5x5 y busca un valor específico. Usa break para salir de ambos ciclos cuando lo encuentres.

## Ejercicios Adicionales
- Practica usando etiquetas (labels) con break y continue
- Investiga alternativas a break y continue para código más limpio
