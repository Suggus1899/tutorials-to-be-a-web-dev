# Día 4: GitHub Workflow

En este día aprenderás a usar GitHub para gestión de proyectos.

## Contenido
- Issues y labels
- Projects (Kanban)
- Milestones
- Documentación

## Tareas

### Tarea 1: Issues
Practica con issues:
- Crear issues descriptivos
- Usar templates de issues
- Asignar labels y milestones
- Vincular PRs a issues

### Tarea 2: GitHub Projects
Crea un project board:
- Columnas: To Do, In Progress, Done
- Arrastra issues entre columnas
- Automatiza movimientos

### Tarea 3: Documentación
Mejora la documentación:
- README.md completo
- CONTRIBUTING.md
- CODE_OF_CONDUCT.md
- CHANGELOG.md

## Ejercicios Adicionales
- Configura issue templates
- Practica con GitHub Wiki
