# Día 7: Práctica Integradora

En este día realizarás ejercicios que integran todos los conceptos de la semana.

## Contenido
- Repaso de Git avanzado
- Repaso de GitHub workflow
- Repaso de GitHub Actions
- Proyecto integrador

## Proyecto del Día

### Proyecto: Repositorio con CI/CD Completo

Configura un repositorio profesional:

1. **Configuración Git:**
   - Estrategia de branching definida
   - Branch protection rules
   - Commit messages convencionales

2. **GitHub Configuration:**
   - README.md completo
   - Issue templates (bug, feature)
   - PR template
   - CONTRIBUTING.md

3. **CI Pipeline:**
   - Lint en cada PR
   - Tests automáticos
   - Build verification
   - Coverage report

4. **CD Pipeline:**
   - Deploy a staging en PRs
   - Deploy a production en merge a main
   - Notificaciones

5. **Project Management:**
   - Project board configurado
   - Labels organizados
   - Milestones para releases
