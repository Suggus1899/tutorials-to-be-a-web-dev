# Día 3: GitHub Pull Requests

En este día aprenderás sobre Pull Requests y code review.

## Contenido
- Crear Pull Requests
- Code reviews
- Resolver conflictos
- Merge strategies

## Tareas

### Tarea 1: Crear un PR
Practica el flujo:
1. Crea rama feature
2. Haz commits
3. Push a GitHub
4. Crea Pull Request
5. Describe los cambios

### Tarea 2: Code Review
Practica revisar código:
- Comentarios constructivos
- Sugerencias de mejora
- Aprobar o solicitar cambios

### Tarea 3: Resolver conflictos
Practica resolviendo conflictos:
- Crea un conflicto intencionalmente
- Resuélvelo localmente
- Push los cambios

## Ejercicios Adicionales
- Configura templates de PR
- Practica con diferentes merge strategies
