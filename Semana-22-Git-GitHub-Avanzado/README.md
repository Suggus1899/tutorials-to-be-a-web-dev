# Semana 22: Git y GitHub Avanzado

## Objetivos de la Semana

Esta semana aprenderás Git avanzado y colaboración:

### 1. Git avanzado
- Branching strategies (Git Flow)
- Rebase vs Merge
- Cherry-pick y stash

### 2. GitHub avanzado
- Pull requests y code reviews
- GitHub Actions (CI/CD)
- Issues y project management

### 3. Colaboración en equipo
- Resolución de conflictos
- Semantic versioning
- Documentación del proyecto

## Estructura de la Semana

- **Día 1-2**: Git avanzado y branching strategies
- **Día 3-4**: GitHub y flujo de trabajo
- **Día 5-6**: CI/CD con GitHub Actions
- **Día 7**: Práctica integradora
