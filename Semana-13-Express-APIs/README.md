# Semana 13: Express y APIs

## Objetivos de la Semana

Esta semana aprenderás a crear APIs REST con Express:

### 1. Introducción a Express
- Instalación y configuración
- Primera aplicación Express
- Middleware básico

### 2. Routing
- Definir rutas
- Parámetros de ruta
- Query parameters

### 3. CRUD básico
- Crear recursos (POST)
- Leer recursos (GET)
- Actualizar recursos (PUT/PATCH)
- Eliminar recursos (DELETE)

## Estructura de la Semana

- **Día 1-2**: Configuración de Express y routing
- **Día 3-4**: Operaciones CRUD
- **Día 5-6**: API REST completa
- **Día 7**: Práctica integradora
