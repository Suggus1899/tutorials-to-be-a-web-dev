# Día 1: Introducción a Express

En este día aprenderás los fundamentos de Express.js.

## Contenido
- Qué es Express.js
- Instalación y configuración
- Primera aplicación Express
- app.listen y app.get

## Tareas

### Tarea 1: Configurar Express
Practica:
- Crea un nuevo proyecto Node.js
- Instala Express con npm install express
- Crea el archivo principal (app.js o index.js)

### Tarea 2: Primer servidor Express
Crea un servidor que:
- Escuche en el puerto 3000
- Tenga una ruta GET en /
- Responda con un mensaje de bienvenida

### Tarea 3: Múltiples rutas
Agrega más rutas:
- /about que retorne información
- /api que retorne JSON
- /status que retorne el estado del servidor

## Ejercicios Adicionales
- Experimenta con res.json() vs res.send()
- Practica con variables de entorno para el puerto
