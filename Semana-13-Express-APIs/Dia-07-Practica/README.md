## Proyecto del Día

### Proyecto: API de Biblioteca

Crea una API para gestionar una biblioteca:

1. **Recursos a implementar:**
   - Libros (id, titulo, autor, isbn, disponible)
   - Autores (id, nombre, nacionalidad)
   - Préstamos (id, libro_id, usuario, fecha_prestamo, fecha_devolucion)

2. **Endpoints para Libros:**
   - CRUD completo
   - Filtrar por autor, disponibilidad
   - Buscar por título

3. **Endpoints para Autores:**
   - CRUD completo
   - Listar libros de un autor

4. **Endpoints para Préstamos:**
   - Registrar préstamo
   - Registrar devolución
   - Listar préstamos activos

5. **Características:**
   - Estructura de proyecto organizada
   - Respuestas consistentes
   - Validación de datos
   - Manejo de errores

## Tareas Adicionales

### Tarea 1: Relaciones
Implementa la relación entre libros y autores.

### Tarea 2: Estadísticas
Agrega un endpoint de estadísticas (libros más prestados, etc.)
