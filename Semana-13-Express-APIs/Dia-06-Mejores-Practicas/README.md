# Día 6: Mejores Prácticas

En este día aprenderás mejores prácticas para APIs Express.

## Contenido
- Estructura de proyecto
- Variables de entorno
- Manejo de errores global
- Logging básico

## Tareas

### Tarea 1: Organizar el proyecto
Crea una estructura de carpetas:
```
src/
  routes/
  controllers/
  utils/
  app.js
```

### Tarea 2: Variables de entorno
Implementa:
- Usa dotenv para cargar .env
- Configura PORT, NODE_ENV
- Crea configuración para diferentes entornos

### Tarea 3: Manejo de errores
Implementa:
- Middleware de error global
- Errores personalizados
- Logging de errores

## Ejercicios Adicionales
- Implementa CORS para permitir peticiones de otros dominios
- Agrega rate limiting básico
