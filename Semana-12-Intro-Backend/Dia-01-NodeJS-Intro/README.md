# Día 1: Introducción a Node.js

En este día aprenderás qué es Node.js y sus fundamentos.

## Contenido
- Qué es Node.js
- V8 engine y event loop
- Node.js vs navegador
- Instalación y configuración

## Tareas

### Tarea 1: Configurar Node.js
Practica:
- Verifica que Node.js esté instalado (node -v)
- Ejecuta tu primer script con node archivo.js
- Usa la REPL de Node (node sin argumentos)

### Tarea 2: Diferencias con el navegador
Explora:
- Objetos disponibles en Node (process, global)
- Objetos NO disponibles (window, document)
- Variables de entorno con process.env

### Tarea 3: Crear proyecto Node.js
Practica:
- Inicializa un proyecto con npm init
- Explora package.json
- Crea scripts en package.json

## Ejercicios Adicionales
- Experimenta con process.argv para argumentos de línea de comandos
- Investiga sobre el event loop de Node.js
