# Día 4: Servidor Básico

En este día crearás un servidor más completo.

## Contenido
- Routing básico
- Manejar diferentes métodos HTTP
- Parsear query strings
- Servir archivos estáticos

## Tareas

### Tarea 1: Router básico
Crea un servidor con rutas:
- / → Página de inicio
- /about → Página "Acerca de"
- /contact → Página de contacto
- /api/users → JSON con usuarios

### Tarea 2: Manejar métodos HTTP
Implementa:
- GET para obtener datos
- POST para recibir datos (parsear body)
- Responder apropiadamente a cada método

### Tarea 3: Query strings
Practica:
- Parsear query strings (?nombre=valor)
- Usar el módulo url de Node
- Filtrar datos según parámetros

## Ejercicios Adicionales
- Implementa un servidor de archivos estáticos
- Crea un logger que registre todas las peticiones
