# Día 2: Módulos CommonJS

En este día aprenderás el sistema de módulos de Node.js.

## Contenido
- require() y module.exports
- Módulos propios
- Módulos de Node.js (built-in)
- npm y módulos de terceros

## Tareas

### Tarea 1: Crear módulos propios
Practica:
- Crea un archivo con funciones de utilidad
- Exporta las funciones con module.exports
- Importa y usa las funciones en otro archivo

### Tarea 2: Exportar de diferentes formas
Practica:
- Exportar un objeto con múltiples propiedades
- Exportar una única función
- Exportar una clase

### Tarea 3: Usar módulos npm
Practica:
- Instala un paquete con npm install
- Usa el paquete en tu código
- Explora node_modules y package-lock.json

## Ejercicios Adicionales
- Investiga la diferencia entre dependencies y devDependencies
- Experimenta con ES Modules (import/export)
