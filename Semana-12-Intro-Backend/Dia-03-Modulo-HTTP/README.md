# Día 3: Módulo HTTP

En este día aprenderás a usar el módulo http de Node.js.

## Contenido
- Crear un servidor HTTP
- Request y Response
- Status codes
- Headers

## Tareas

### Tarea 1: Servidor básico
Crea un servidor que:
- Escuche en el puerto 3000
- Responda con "Hola Mundo"
- Muestre en consola cuando recibe una petición

### Tarea 2: Diferentes respuestas
Modifica el servidor para:
- Responder diferente según la URL (req.url)
- Devolver HTML para la página principal
- Devolver JSON para /api

### Tarea 3: Status codes y headers
Practica:
- Responder con diferentes status codes
- Agregar headers personalizados
- Responder con Content-Type apropiado

## Ejercicios Adicionales
- Crea una página 404 personalizada
- Experimenta con diferentes métodos HTTP (req.method)
