# Semana 12: Introducción al Backend

## Objetivos de la Semana

Esta semana aprenderás los fundamentos de Node.js:

### 1. Node.js y módulos
- Qué es Node.js
- Sistema de módulos CommonJS
- require() y module.exports

### 2. Módulo http
- Crear un servidor HTTP
- Manejar peticiones y respuestas
- Status codes

### 3. Módulo fs (File System)
- Leer archivos
- Escribir archivos
- Operaciones síncronas vs asíncronas

## Estructura de la Semana

- **Día 1-2**: Node.js y sistema de módulos
- **Día 3-4**: Módulo http y servidores
- **Día 5-6**: Módulo fs y manejo de archivos
- **Día 7**: Práctica integradora
