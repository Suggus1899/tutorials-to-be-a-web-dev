# Día 5: Módulo FS (File System)

En este día aprenderás a trabajar con archivos en Node.js.

## Contenido
- Leer archivos (readFile, readFileSync)
- Escribir archivos (writeFile, writeFileSync)
- Operaciones síncronas vs asíncronas
- Encoding y buffers

## Tareas

### Tarea 1: Leer archivos
Practica:
- Lee un archivo de texto con readFileSync
- Lee un archivo con readFile (callback)
- Lee un archivo con fs.promises (async/await)

### Tarea 2: Escribir archivos
Practica:
- Escribe un archivo nuevo
- Sobrescribe un archivo existente
- Agrega contenido a un archivo (appendFile)

### Tarea 3: Casos prácticos
Implementa:
- Leer un archivo JSON y parsearlo
- Guardar datos como JSON
- Copiar contenido de un archivo a otro

## Ejercicios Adicionales
- Experimenta con readdir para listar archivos de un directorio
- Practica con stat para obtener información del archivo
