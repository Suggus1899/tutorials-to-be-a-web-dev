# Día 6: FS Avanzado

En este día aprenderás operaciones avanzadas con archivos.

## Contenido
- Crear y eliminar directorios
- Verificar existencia de archivos
- Streams para archivos grandes
- Watch para cambios

## Tareas

### Tarea 1: Operaciones con directorios
Practica:
- Crear un directorio (mkdir)
- Crear directorios anidados (recursive: true)
- Eliminar directorio (rmdir, rm)

### Tarea 2: Verificar y gestionar
Practica:
- Verificar si un archivo existe
- Obtener información del archivo (stat)
- Renombrar y mover archivos

### Tarea 3: Streams
Practica:
- Leer archivo grande con createReadStream
- Escribir con createWriteStream
- Pipe entre streams

## Ejercicios Adicionales
- Implementa un observador de cambios en archivos (fs.watch)
- Crea una utilidad de backup de archivos
