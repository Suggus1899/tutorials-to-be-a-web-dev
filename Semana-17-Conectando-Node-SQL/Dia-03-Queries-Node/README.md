# Día 3: Queries desde Node.js

En este día aprenderás a ejecutar queries SQL desde Node.js.

## Contenido
- Ejecutar SELECT, INSERT, UPDATE, DELETE
- Manejar resultados
- Interpolación segura
- Manejo de errores

## Tareas

### Tarea 1: SELECT queries
Practica consultando:
- Obtener todos los registros
- Obtener un registro por ID
- Filtrar con parámetros

### Tarea 2: INSERT, UPDATE, DELETE
Practica modificando datos:
- Insertar y obtener el lastID
- Actualizar y verificar changes
- Eliminar registros

### Tarea 3: Integrar con Express
Conecta la BD con una API:
- GET /api/users → SELECT * FROM users
- POST /api/users → INSERT INTO users
- PUT /api/users/:id → UPDATE users
- DELETE /api/users/:id → DELETE FROM users

## Ejercicios Adicionales
- Implementa paginación con LIMIT y OFFSET
- Practica con búsqueda usando LIKE
