# Semana 17: Conectando Node.js con SQL

## Objetivos de la Semana

Esta semana aprenderás a integrar bases de datos con Node.js:

### 1. Drivers de bases de datos
- sqlite3 para Node.js
- Instalación y configuración
- Conexión a la base de datos

### 2. Queries desde Node.js
- Ejecutar queries
- Manejo de resultados
- Prepared statements

### 3. Repository Pattern
- Separar lógica de datos
- Crear repositorios
- Mejores prácticas

## Estructura de la Semana

- **Día 1-2**: Drivers y conexión
- **Día 3-4**: Queries y manejo de datos
- **Día 5-6**: Repository pattern
- **Día 7**: Práctica integradora
