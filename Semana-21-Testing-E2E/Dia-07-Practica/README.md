- Repaso de interacciones

- Proyecto integrador

## Proyecto del Día

### Proyecto: Suite E2E para Aplicación Web

Crea una suite E2E completa:

1. **Flujos a testear:**
   - Registro de usuario
   - Login y logout
   - CRUD de recursos (crear, ver, editar, eliminar)
   - Navegación entre páginas

2. **Tests específicos:**
   - Happy path: flujo exitoso completo
   - Edge cases: errores, validaciones
   - Responsive: diferentes tamaños de pantalla

3. **Organización:**
   - Page Objects para cada página
   - Fixtures para datos de prueba
   - Custom commands reutilizables
   - Interceptores para API

4. **CI/CD:**
   - Configuración para GitHub Actions
   - Screenshots y videos
   - Reportes de resultados
