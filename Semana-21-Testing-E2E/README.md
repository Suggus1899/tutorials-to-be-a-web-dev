# Semana 21: Testing End-to-End

## Objetivos de la Semana

Esta semana aprenderás a realizar tests E2E de aplicaciones web:

### 1. Introducción a E2E Testing
- Qué es testing E2E
- Cypress vs Playwright
- Configuración inicial

### 2. Escribir tests E2E
- Selectores y navegación
- Interacciones con la UI
- Assertions y esperas

### 3. Tests avanzados
- Fixtures y mocking
- CI/CD integration
- Mejores prácticas

## Estructura de la Semana

- **Día 1-2**: Introducción y configuración de herramientas E2E
- **Día 3-4**: Escribir tests básicos
- **Día 5-6**: Tests avanzados y CI/CD
- **Día 7**: Práctica integradora
