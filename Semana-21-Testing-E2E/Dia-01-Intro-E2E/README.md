# Día 1: Introducción a E2E Testing

En este día aprenderás los fundamentos del testing end-to-end.

## Contenido
- Qué es testing E2E
- Diferencia con otros tipos de tests
- Herramientas populares
- Cuándo usar E2E

## Tareas

### Tarea 1: Conceptos E2E
Estudia y documenta:
- Testing E2E simula un usuario real
- Prueba el sistema completo
- Más lento pero más realista

### Tarea 2: Comparar herramientas
Investiga:
- Cypress: popular, fácil de usar
- Playwright: multi-browser, moderno
- Puppeteer: Google, headless
- Ventajas de cada uno

### Tarea 3: Cuándo usar E2E
Analiza:
- Flujos críticos de usuario
- Smoke tests
- Regression testing
- No para todo (pirámide de testing)

## Ejercicios Adicionales
- Revisa documentación de Cypress y Playwright
- Decide qué herramienta usar para tu proyecto
