# Día 1: Introducción a SQL

En este día aprenderás los fundamentos de SQL y bases de datos relacionales.

## Contenido
- Qué es SQL
- Bases de datos relacionales
- Tablas, filas y columnas
- RDBMS populares

## Tareas

### Tarea 1: Conceptos básicos
Estudia y documenta:
- Qué es una base de datos
- Diferencia entre SQL y NoSQL
- Qué es un RDBMS
- Por qué usar bases de datos

### Tarea 2: Diseño conceptual
Diseña en papel una base de datos para:
- Una lista de tareas (id, título, completada)
- Un blog (posts, autores)
- Una tienda (productos, categorías)

### Tarea 3: Vocabulario SQL
Familiarízate con términos:
- Table, Row, Column
- Primary Key, Foreign Key
- Query, Statement

## Ejercicios Adicionales
- Investiga las diferencias entre SQLite, MySQL, PostgreSQL
- Dibuja diagramas ER (Entidad-Relación) simples
