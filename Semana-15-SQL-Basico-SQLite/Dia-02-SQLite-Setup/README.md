# Día 2: SQLite Setup

En este día configurarás SQLite para trabajar con bases de datos.

## Contenido
- Qué es SQLite
- Instalación de SQLite
- Herramientas para SQLite
- Crear primera base de datos

## Tareas

### Tarea 1: Instalar SQLite
Practica:
- Instala SQLite en tu sistema
- Verifica la instalación (sqlite3 --version)
- Explora las opciones de línea de comandos

### Tarea 2: Herramientas visuales
Instala una herramienta visual:
- DB Browser for SQLite (recomendado)
- O extensión de VS Code para SQLite
- Familiarízate con la interfaz

### Tarea 3: Primera base de datos
Crea tu primera base de datos:
- Usa sqlite3 nombrebd.db
- Explora comandos .help, .tables
- Sal con .exit

## Ejercicios Adicionales
- Practica con comandos de metadatos (.schema, .dump)
- Exporta e importa datos básicos
