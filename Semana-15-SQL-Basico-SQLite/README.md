# Semana 15: SQL Básico y SQLite

## Objetivos de la Semana

Esta semana aprenderás los fundamentos de bases de datos relacionales:

### 1. Introducción a SQL
- Qué es SQL
- Bases de datos relacionales
- SQLite y configuración

### 2. Tablas
- Crear tablas (CREATE TABLE)
- Tipos de datos
- Constraints (PRIMARY KEY, NOT NULL, etc.)

### 3. Operaciones básicas
- SELECT: consultar datos
- INSERT: insertar registros
- UPDATE y DELETE

## Estructura de la Semana

- **Día 1-2**: Introducción a SQL y SQLite
- **Día 3-4**: Crear tablas y tipos de datos
- **Día 5-6**: SELECT, INSERT, UPDATE, DELETE
- **Día 7**: Práctica integradora
