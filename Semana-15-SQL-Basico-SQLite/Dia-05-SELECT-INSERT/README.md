# Día 5: SELECT e INSERT

En este día aprenderás a consultar e insertar datos.

## Contenido
- INSERT INTO para agregar datos
- SELECT para consultar
- WHERE para filtrar
- ORDER BY para ordenar

## Tareas

### Tarea 1: INSERT básico
Practica insertando datos:
```sql
INSERT INTO usuarios (nombre, email) VALUES ('Juan', 'juan@email.com');
```
- Inserta múltiples registros
- Inserta especificando todas las columnas

### Tarea 2: SELECT básico
Practica consultando:
- SELECT * para todas las columnas
- SELECT nombre, email para columnas específicas
- Alias con AS

### Tarea 3: WHERE y ORDER BY
Filtra y ordena:
- WHERE nombre = 'Juan'
- WHERE precio > 100
- ORDER BY nombre ASC/DESC
- LIMIT para limitar resultados

## Ejercicios Adicionales
- Practica con LIKE para búsquedas parciales
- Experimenta con AND, OR en WHERE
